## 1. 제안 개요

### 1-1. 구상 및 개발 배경

**두 흐름의 충돌이 한국 에너지 정책의 가장 시급한 과제다.**

```
[흐름 1] AI 시대 전력 수요 폭증
  - 한국 데이터센터 전력 수요 : 2025년 4,461MW → 2028년 6,175MW (연 +11%)
  - 2038년 예측 : 2025년 대비 약 6배
  - 신규 전력 신청 67%가 수도권 집중

[흐름 2] 재생에너지 비중 급증과 변동성 확대
  - 2025.4 화석연료 발전 비중 49.5% (사상 첫 50% 미만)
  - 태양광 비중 9.2% 사상 최고
  - 2040년 출력제한량 예상 7.8~23.4 TWh (서울시 1년 사용량의 1/3)
```

**충돌의 본질** : AI 데이터센터(24시간 일정한 대량 수요) vs 재생E(시간·날씨별 변동 공급) → 화석연료 발전소가 변동성을 메우는 역설 발생.

해법은 두 가지 — ① 공급 측(발전소·ESS 추가, 막대한 비용·시간) vs ② **수요 측(가정 시간 분산, 거의 무료·즉시 가능).** 그러나 **시민에게 "지금이 청정 시간"임을 알려주는 도구가 한국에 없다.**

### 1-2. 결정적 정책 공백

| 제도 | 참여 가구 | 보상 단가 |
|---|---|---|
| 한전 에너지캐시백 (월 절약형) | **166만 호** | 30~100원/kWh |
| 국민DR / 에너지쉼표 (시간 분산형) | **7,430호** (캐시백의 0.45%) | **약 1,300원/kWh** |

> 보상 단가가 **13~43배 더 높은데도** 참여율은 1/200. 원인 — *알기 어렵고, 참여하기 어렵고, 효과를 모른다.* AMI 인프라(2,005만 호 보급)와 시민 참여 의지는 있는데 **시민-DR을 매개하는 똑똑한 인터페이스가 없다.**

### 1-3. 목표

1. AMI × 발전 믹스 × 탄소 배출계수를 AI로 융합해 **"청정 시간"을 시민에게 선제 알림**
2. 한전 캐시백·국민DR·탄소중립포인트·에너지바우처 **4개 기존 제도를 자동 통합 정산**
3. 가정 DR 참여율을 **0.45% → 5% 이상**으로 확대 (LNG 1기 회피 규모)

### 1-4. 대상

- **1차 잠재 사용자** : AMI 보급 가구 2,005만 호 전체
- **우선 진입 페르소나** : 맞벌이 가구(자동 모드 수요), EV·태양광 보유 가정(유연 부하 大), 아파트 관리사무소(B2B), 에너지바우처 수급자(자동 옵트인)

### 1-5. 기대 효과

| 지표 | 효과 |
|---|---|
| 피크 수요 감축 | -1GW (가정 5% 시간 분산 시) → **LNG 화력 1기(약 5천억원) 건설 회피** |
| 재생E 출력제한 감소 | -2TWh / 년 = 200만 가구 1년 사용량 회수 |
| CO₂ 감축 | **약 100만 톤 / 년** |
| 가구당 보상 | 캐시백·DR·포인트 합산 **연 5~10만원 자동 적립** |

### 3-1. 기존 서비스 한계

| 서비스 | 한계 |
|---|---|
| My한전 | 사용량 조회·비교만, 시간 이동 가이드 부재 |

| 국민DR / 에너지쉼표 | 발령 시점만 통보, 신청·정산 수동 → 가정 7,430호 정체 |
| 서울 에너지 지도 | 동/구 단위 통계, 가구 행동 가이드 없음 |
| 포스코 에너지 쉼표 챌린지 | 단지 단위 단발성 이벤트, AI 매개 없음 |

### 3-2. 청정 시간의 6가지 차별점

1. **행동 인식 전환** : "절약(덜 써라)" → **"시간 이동(다른 시간에 써라)"** 으로 시민 부담 최소화 → 참여 폭증 가능
2. **한국 최초 가정용 Carbon-Aware Computing** : Microsoft Azure·Google이 데이터센터에 적용한 기술을 가정에 이식
3. **NILM 기반 가전별 분석** : 한국에서 EDF/NILMFormer 수준 가정 분석 서비스 부재 → 첫 도입
4. **4개 제도 자동 통합** : 한전 캐시백 + 국민DR + 탄소중립포인트 + 에너지바우처를 단일 인터페이스로 자동 신청·정산 (신규 제도 신설 0개)
5. **24시간 사전 청정 시간 공시** : 다른 서비스는 발령 시점만, 우리는 미래 24시간 사전 알림
6. **자동 모드** : 스마트 가전 연결 시 사용자 개입 없이 시간 이동 자동 실행

### 3-3. 해외 검증 (벤치마크)

| 사례 | 검증 내용 |
|---|---|
| **Octopus Agile (영국)** | 30분 단위 동적 요금 → 시민이 자발적으로 가전 시간 이동, 가구당 연 30~50% 절감 |
| **OhmConnect (미국)** | 캘리포니아·텍사스 약 200,000 가구, 한 주 1GWh 감축, 가정 DR을 도매시장 자원화 |
| **EDF / NILMFormer (프랑스)** | KDD 2025 SOTA 모델을 EDF가 자사 모니터링 백본으로 채택, 수백만 가구 실서비스 |
| **WattTime / Electricity Maps** | 5분 단위 한계 배출계수 API 표준화, Microsoft·Google Cloud 채택 |

> 한국은 모든 인프라·데이터·법제(분산E 특별법 2024.6 시행)가 갖춰져 있는데 **통합 시민 매개 서비스만 0개.** 청정 시간이 그 빈자리를 정확히 메운다.

## 4. 구체성

### 4-1. 사용자 여정 (5단계)

```
[Step 1] 가입 (5분)
  - 한전 마이데이터 1회 인증
  - 가정 정보 입력 : 평수 · 가구원 수 · 가전 보유 · EV/태양광/ESS 여부

[Step 2] 학습 (1주일)
  - AI가 가정 사용 패턴 학습
  - NILM으로 가전별 분리
  - 라이프스타일 페르소나 자동 분류 (밤형/아침형, 재택/외근)

[Step 3] 청정 시간 알림 시작
  - 매일 아침 : "오늘의 청정 시간 TOP 3"
  - 실시간 푸시 : "지금 풍력 폭주 → 세탁기 권장"
  - 회피 알림 : "오후 6~9시 LNG 피크 → 큰 가전 자제"

[Step 4] 행동 + 자동 보상
  - 알림 따른 행동 → 캐시백·DR·포인트 자동 적립
  - 스마트 세탁기·식기세척기 보유 시 : 빨래·그릇 세팅 → 앱이 시작 시간 예약 자동 설정
  - EV·전기 온수기·로봇청소기·ESS : 완전 자동 시간 이동
  - 주간 리포트 : "탄소 -X kg, 보상 +Y원"

[Step 5] 누적 효과 가시화
  - 연간 누적 환경 기여 ("소나무 OO그루 1년치")
  - 지역·전국 순위, 월간 챌린지
```

### 4-2. 4대 핵심 기능

#### ① 청정 시간 알림 (Green Hour Push)

```python
for each user:
    user_profile = NILM(user.AMI_LP_data)
    flexible_load = identify_flexible(user_profile)  # 세탁기·식기세척기·EV 등
    forecast = EPSIS_24h_forecast()
    carbon_intensity = calc_carbon_per_hour(forecast)
    optimal_hours = top_3(carbon_intensity, low_first)
    push("내일 11~13시 태양광 정점 — 세탁기 사용 시 탄소 -68%")
```

알림 예시 :
- "오늘 13~15시 태양광 정점, 세탁기 시간 이동 시 탄소 -68%"
- "내일 새벽 2~4시 풍력 강풍대, 식기세척기 / EV 충전 추천"
- "오후 7시 저녁 피크, 큰 가전 30분 미루면 8% 절약"

#### ② 스마트 예약 (Smart Schedule) — 가전별 3단계 자동화

가전마다 자동화 수준이 다름. 솔직한 분류 :

| 단계 | 대상 가전 | 사용자 개입 | 메커니즘 |
|---|---|---|---|
| **알림형** (기본·90%) | 모든 일반 가전 (대다수 한국 가정) | 사용자가 직접 가전 조작 | 알림에 따라 사용자가 시간 옮김 |
| **예약 자동화** | 스마트 세탁기·식기세척기·건조기 (Samsung SmartThings, LG ThinQ API) | 빨래·그릇·세제 미리 세팅 | 앱이 가전 API로 시작 시간 자동 예약 |
| **완전 자동** | EV 충전, 전기 온수기, 로봇청소기, ESS·태양광 인버터 | 0 (케이블 꽂기·예약 청소 등 표준 기능) | 사용자 개입 없이 시간 이동 |

```
[예약 자동화 흐름 — 스마트 세탁기 예시]
사용자  : 퇴근 후 빨래·세제 세팅 + "자동 모드 ON, 시간 창 = 18~익일 9시"
       ↓
AI 선택 : 시간 창 안에서 가장 깨끗하고 저렴한 시간 선택 (예: 새벽 2시)
       ↓
가전 API: SmartThings/ThinQ 예약 시작 명령 자동 전송
       ↓
완료 보고: "세탁 완료 — 평소 대비 탄소 -52%, 비용 -180원"
```

> **한국 가정 90% 이상이 일반 가전이므로 알림형이 메인 채널.** 스마트 가전·EV 보유 가정의 확대에 따라 예약 자동화·완전 자동 비중이 점진 증가하는 구조 (영국 Octopus도 동일 단계 거쳐 확산).

#### ③ 국민DR 자동 참여 (DR Auto-Join)

```
DR 발령 (전력거래소) : "오후 2~5시 감축 시 1,300원/kWh 보상"
       ↓
앱 푸시 : "[국민DR 발령] 가전 자제 시 5,200원 적립" → 1탭 참여
       ↓
실제 감축량을 AMI로 측정 → 자동 정산 → 다음달 요금 차감
```
> 현재 가정 7,430호 정체 원인이 **개별 신청·관리 부담** → 자동화 시 10만 호 단위 참여 가능.

#### ④ 가시화 + 게임화 (Impact Dashboard)

```
주간 리포트 :
  "이번 주 청정 시간 참여 : 12회
   탄소 감축      : 14.3 kg = 소나무 2.1그루 1년치
```

## 6. 사회 문제 해결

본 서비스가 직접 해결하는 사회 문제 6가지.

### 6-1. AI 시대 전력 수요 폭증과 탄소중립 위협
- 데이터센터 2038년까지 6배 증가, 화석연료 의존 회귀 위험
- **해법** : 가정 부문 시간 분산이 가장 빠르고 저렴한 흡수 수단

### 6-2. 재생에너지 출력제한 (자원 낭비)
- 2040년 23.4 TWh 출력제한 예상 (서울시 1년 사용량의 1/3)
- **해법** : 청정 시간 알림으로 가정이 잉여 재생E 흡수 → 출력제한 감소

### 6-3. 가정 DR 참여 정책 공백
- 캐시백 166만 호 vs 국민DR 7,430호 (참여율 0.45%)
- 보상 단가 13~43배 더 높은데도 1/200 참여
- **해법** : 자동 알림·자동 참여·자동 정산으로 진입 장벽 0

### 6-4. 에너지 복지 사각지대
- 에너지바우처 수급자의 잔액·소진 관리 어려움
- **해법** : 자동 옵트인 + 잔액 실시간 알림 + 캐시백 자동 적립으로 디지털 격차 해소

### 6-5. 시민의 환경 행동 동기 부족
- 추상적 "탄소중립" 메시지로는 행동 변화 한계
- **해법** : 구체적 행동(시간 이동) + 즉시 보상 + 게임화로 지속 참여

### 6-6. 가정 부문 에너지 정보 비대칭
- 시간대별 탄소 격차 정보가 시민에게 미공개
- **해법** : 청정 시간 공시 + 시간대별 탄소 배출계수 공시 의무화 정책 제언
