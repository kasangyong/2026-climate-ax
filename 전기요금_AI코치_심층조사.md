# 우리 집 전기요금 AI 코치 — 심층 조사 노트

> 2026 기후부 AX 아이디어 경진대회 (사업·정책 아이디어 기획 / 에너지효율 미션)
> 기준일: 2026.05.09

---

## 1. 왜 이 주제인가 — 문제의 핵심

한국에서 2024년 한 해에 일어난 두 가지 사실이 있다.

1. **한전이 AMI(스마트미터) 전국 2,005만 호 보급을 완료했다.**
2. **2024년 8월, 1,000만 가구 이상이 누진 3단계 요금 폭탄을 맞았다.**

이 두 사실은 같은 인프라 위에서 일어났다. 모든 가구가 15분 단위로 자기 전력 사용량 데이터를 갖고 있다. 그런데 그 데이터를 활용해 "당신은 지금 위험합니다", "이렇게 하면 5만원 아낄 수 있습니다"라고 알려주는 서비스는 한국에 없다.

> 인프라는 완성됐다. **활용이 0이다.** 이게 이 아이디어의 출발점이다.

## 2. 한국 가정 전기요금 — 충격적 수치들

### 2024년 여름 전기요금 폭탄

| 지표 | 2024년 8월 | 전년 대비 |
|------|-----------|---------|
| 가구당 평균 전력 사용량 | 363kWh | **+9%** |
| 가구당 평균 전기요금 | 63,610원 | **+13% (+7,520원)** |
| 누진 3단계 적용 가구 수 | **1,000만+** | 전체의 **40%+** |

핵심 포인트: 사용량은 9% 늘었는데 요금은 13% 늘었다. **누진제 때문에 일정 구간을 넘으면 요금이 비대칭적으로 폭증**한다. 그런데 한 달이 끝나야 알 수 있다. 미리 알 수 있는 도구가 없다.

### 2025년 7~8월 누진 구간 한시 완화

정부가 급등하는 전기요금 부담 때문에 누진 구간을 한시적으로 완화했다. **이는 "누진제가 가구에 실질적 부담"이라는 정부의 공식 인정**이다.

### 가구 구성 변화

- 1~2인 가구가 **과반수**, 3인 이하 가구가 **74.8%**
- 누진제는 1989년 4인 가구 기준으로 설계됐는데, 지금은 1인 가구가 대세
- "가구당 평균 사용량"으로 누진을 매기지만, 실제 가구 구성이 달라졌다는 구조적 모순

### 가정용 전기요금 누진제 구조 (2025 여름)

| 단계 | 사용 구간 | 단가 (kWh당) |
|------|---------|------------|
| 1단계 | 300kWh 이하 | 120원 |
| 2단계 | 301~450kWh | 214.6원 |
| 3단계 | 450kWh 초과 | **307.3원** (1단계의 약 2.6배) |

> 1단계와 3단계의 단가 차이가 2.6배. 이 정보를 모르고 사용하다 월말에 폭탄을 맞는 게 매년 반복된다.

## 3. 인프라 vs. 활용의 결정적 갭

### AMI 보급 현황

- 한전: 2024년까지 **2,005만 호 보급 완료**
- 한전 보유 데이터: **189개 사내 시스템 + 262종 공공데이터 통합, 일일 34억 건 처리, 771TB 빅데이터 실시간 처리**
- 스마트그리드 확산사업 **2026~2030 5년 연장** 결정
- 마이데이터 기반 에너지 종합 플랫폼 추진 중

### My한전/한전ON 앱의 한계 (사용자 후기 기반)

검색으로 발견된 실제 사용자 평가:

| 문제 | 사용자 후기 |
|------|-----------|
| 데이터 정확성 | "로그인 후 우리 집과 다른 집의 데이터가 함께 표시됨" |
| 개선 부재 | "오랫동안 사용했지만 몇 년 동안 개선된 점이 없음" |
| 안정성 | "세션 만료 알림 반복, 원인 모를 오류" |
| 분석 부재 | 월간 총 사용량 숫자만 표시. 분석/가이드 없음 |
| 가전별 정보 | **0** — 어떤 가전이 얼마나 쓰는지 알 수 없음 |
| 누진 경보 | **없음** — 월말이 돼야 단계 진입 확인 가능 |

### 결론적 갭

```
한전은 데이터를 가지고 있다 (2,005만 호 × 96개/일 = 일 19억 건)
한전은 AI 플랫폼도 가지고 있다 (771TB 빅데이터 실시간 처리)
한전 앱도 있다 (My한전, 한전ON, 파워플래너)

→ 그런데 사용자는 "이번 달 342kWh 썼습니다" 한 줄만 본다
→ 왜 많이 썼는지, 어떤 가전이 문제인지, 어떻게 줄일지 모른다
→ 이게 정책 공백이고 시장 공백이다
```

## 4. 핵심 기술 — NILM (Non-Intrusive Load Monitoring)

### NILM이란

**스마트미터 총 전력 데이터만으로 가전별 사용 패턴을 AI로 역추정하는 기술.**

각 가전은 고유한 "전력 시그니처"를 갖는다:
- 냉장고: 일정 주기(20~40분)로 ON/OFF 반복, 100~200W
- 에어컨: 시동 시 큰 피크, 압축기 주기적 ON/OFF, 1~2kW
- 세탁기: 모터 → 가열 → 헹굼 → 탈수 단계별 명확한 전력 패턴
- TV/가전 대기전력: 상시 5~30W 일정 유지

이 패턴들을 AI(주로 Transformer, CNN, LSTM)가 학습해서 분리한다.

### 2025년 최신 SOTA: NILMFormer

**KDD 2025에서 발표된 Transformer 기반 모델** (arXiv:2506.05880)

핵심 혁신:
- Subsequence stationarization/de-stationarization으로 데이터 분포 변화에 강건
- 위치 인코딩에 timestamp 정보만 사용 (도메인 적응성)
- **이미 EDF(프랑스 전력공사)가 자사 모니터링 서비스 백본으로 채택, 수백만 가구에 실서비스 중**

> **이건 결정적이다**: 프랑스 국영 전력회사가 실제 운영 중인 SOTA 모델이 오픈된 논문에 있다. 한국엔 없다.

### 추가 최신 연구 동향

| 논문/모델 | 핵심 기여 | 연도 |
|---------|---------|------|
| NILMFormer | Transformer + non-stationarity 처리, EDF 실서비스 | 2025 (KDD) |
| Active Learning NILM | 라벨 데이터 99% 절감, 정확도 유지 | 2025 |
| Transformer Attention NILM (MDPI) | Temporal pooling 결합, 성능 향상 | 2024 |
| Edge NILM | 엣지 디바이스 추론 최적화 (게이트웨이/홈 허브 적용 가능) | 2025 |
| Semi-supervised NILM | 적은 데이터로도 학습 가능 | 2023 |

### 활용 가능한 오픈소스

| 레포 | 기술 스택 | 용도 |
|------|---------|------|
| [nilmtk/nilmtk](https://github.com/nilmtk/nilmtk) | Python | NILM 표준 툴킷, 알고리즘 비교용 |
| [Virtsionis/torch-nilm](https://github.com/Virtsionis/torch-nilm) | PyTorch | NILMTK 호환 딥러닝 툴킷 |
| [Ming-er/NeuralNILM_Pytorch](https://github.com/Ming-er/NeuralNILM_Pytorch) | PyTorch | SOTA 알고리즘 PyTorch 구현 |
| [OdysseasKr/online-nilm](https://github.com/OdysseasKr/online-nilm) | TensorFlow | 실시간 슬라이딩 윈도우 NILM |

> **중요**: 모델은 다 오픈돼 있다. 우리가 만들 새로운 가치는 **모델 자체가 아니라, 한국 가정 데이터로 미세조정 + 한국 요금제/정책과 통합한 서비스 레이어**다.

## 5. 해외 상용 사례 — 이미 시장이 검증됐다

### Sense (미국)

- 분전반에 클램프 센서 부착 + NILM AI
- **초당 100만 샘플 분석**, 실시간 가전 인식
- 가구당 약 $300 (B2C 하드웨어 + 앱)
- 시장 검증: 미국 대형 유틸리티들과 파트너십

### Smappee (벨기에)

- NILM + 서브미터링 클램프
- **평균 5개 가전 자동 인식, 정확도 70%**
- 상업용은 **구독 모델** (월 단위 SaaS)
- API 개방: 외부 서비스 통합 가능

### EDF (프랑스 전력공사) — 가장 중요한 벤치마크

- NILMFormer 백본 도입
- **수백만 가구에 가전별 사용량 분석 서비스 제공 중**
- 즉, 국영 전력회사가 NILM 기반 개인화 서비스를 이미 한다는 뜻
- **한국은 한전이 같은 데이터를 가지고도 안 한다** → 정책적 공백 명확

### Octopus Energy (영국)

- 시간대별 동적 요금제 + 자동 전기차 충전 최적화
- "Octopus Agile": 30분 단위 요금 변동 → 앱이 알아서 최적 시간 추천
- 영국 에너지 시장 점유율 폭발적 성장

### 한국 시장 현황 = 공백

해외는 NILM + 동적 요금제 + 개인화 코칭이 이미 상용화 단계.
한국은 하드웨어 인프라(AMI)는 갖췄지만 **소프트웨어 서비스 레이어가 비어있음.**

## 6. 한국 정책 환경 — 우리 아이디어와 맞물리는 흐름

### 2026 정부 에너지 정책 방향 (검색 결과 기반)

> "2026년 정책의 핵심은 '요금을 올릴 것인가'가 아니라 **'가격 신호를 어떻게 설계할 것인가'**"

이게 정확히 우리 아이디어가 풀려고 하는 문제다.

| 2026 정부 추진 정책 | 우리 아이디어와의 관계 |
|------------------|-------------------|
| 시간대별 요금제 (TOU) 확대 | 우리 서비스가 시간대별 최적 사용 가이드 → **실효성 확보** |
| 동적/피크 요금제 도입 | 실시간 가격 신호를 사용자 행동으로 연결 |
| 지역별 요금제 | 지역별 데이터 분석 가능한 플랫폼 필요 |
| 수요반응(DR) 시장 확대 | 가정 부문 DR 참여를 우리 앱이 매개 |
| 분산에너지·VPP·V2G | 가구 단위 유연성 자원 관리 도구 필요 |

### 한전 자체 행보

- 스마트그리드 확산사업 **2026~2030 연장** 발표
- 마이데이터 기반 에너지 종합 플랫폼 추진 (서울도시가스 등과 협업)
- 즉, **한전이 직접 이 영역에 뛰어들고 있다 = 시장이 이미 움직이고 있다**

### 기존 정부 인센티브 제도 (활용 가능)

| 제도 | 내용 | 우리 서비스와의 연계 |
|------|------|------------------|
| **탄소중립포인트(에너지)** | 전기·가스·수도 5% 이상 절감 시 포인트 → 현금 교환 | 우리 앱이 자동 산정·신청 대행 |
| **에너지캐시백** | 한전 기존 제도, 절약량에 따라 요금 환급 | 캐시백 추적 + 최적화 |
| **고효율가전 환급** | 에너지공단 효율등급 1등급 가전 구매 시 환급 | 가전 교체 ROI 계산에 통합 |
| **에너지바우처** | 1인 29.5만원/4인 70.1만원 (2025 기준) | 취약계층 잔액 알림 + 절약 가이드 |

→ 우리 서비스는 **신규 제도를 만들 필요 없이, 흩어진 기존 제도를 한 앱으로 묶기**만 해도 가치가 있다.

## 7. 서비스 상세 설계

### 7-1. 데이터 파이프라인

```
[사용자 동의]
  한전 마이데이터 연결 (15분 단위 LP 데이터)

         ↓

[데이터 수집·전처리]
  실시간 스트림 (Kafka or 한전 EDS API 폴링)
  결측 보정, 이상치 제거, 정규화

         ↓

[NILM AI 분리]
  Pre-trained NILMFormer + 한국 가정 데이터 미세조정
  → 가전별 시계열 출력 (에어컨, 냉장고, 세탁기, 보일러, TV, 기타)

         ↓

[분석 엔진]
  - 누진 구간 트래킹 (현재 사용량 vs. 단계 임계치)
  - 월말 예상 요금 시뮬레이션
  - TOU 단가 적용한 시간대 최적화
  - 이상 패턴 감지 (가전 고장 조기 발견)
  - 유사 가구 벤치마크 (평수 + 가구원 수 + 지역)
  - 탄소 배출량 환산 (전력 배출계수 0.4434 tCO₂/MWh)

         ↓

[출력 레이어]
  - Push 알림 (위험 시점 사전 경보)
  - 일간/주간/월간 리포트
  - 행동 가이드 (구체적이고 즉시 실행 가능)
  - 탄소중립포인트 자동 신청
```

### 7-2. 4대 핵심 기능

#### ① 누진 경보 시스템 (가장 핵심)

```
실시간 추세 분석:
  현재 누적 사용량: 312kWh
  남은 일수: 8일
  과거 평균 일 사용량: 14.5kWh
  → 월말 예상: 312 + (14.5 × 8) = 428kWh

판정:
  → 2단계 (~450kWh) 안에서 끝남. 안전.

if 위험:
  Push 알림: "지금 추세라면 22일에 3단계 진입 (월말 487kWh 예상).
              남은 8일 평균 17kWh 이하 유지하면 2단계 마무리 가능.
              구체적 행동:
              - 에어컨 27→28도 (일 -1.2kWh)
              - 세탁 횟수 주 4→3회 (일 -0.4kWh)
              - 대기전력 차단 (일 -0.6kWh)
              총 절약: 17.6kWh/일 → 3단계 진입 회피
              예상 절약: 31,200원"
```

#### ② TOU 시간대 최적화

```
사용자 패턴 분석:
  세탁기 평균 사용 시간: 평일 오후 3시 (중간부하 단가 110원/kWh)
  
제안:
  "세탁을 밤 11시~새벽 9시로 옮기면
   경부하 단가(60원/kWh) 적용
   월 8회 × 1.2kWh × (110 - 60) = 480원 절약 → 연 5,760원
   
   에어컨 잠금 예약: 새벽 1~5시는 경부하 구간
   여름 한 달 야간 사용 비중 30%로 늘리면 월 3,800원 절약"
```

#### ③ 가전 이상 감지

```
NILM 출력으로 가전별 평균 패턴 학습 (지난 3개월)

탐지 케이스:
  - 냉장고 대기 전력 평소 80W → 갑자기 130W 지속
    → "냉장고 문이 제대로 닫혔는지 확인. 또는 냉매 부족 가능성"
  - 보일러 가동 시간 평소 일 2시간 → 일 5시간
    → "단열 문제 또는 보일러 효율 저하. 점검 권고"
  - 에어컨 압축기 주기가 비정상적으로 짧아짐
    → "필터 청소 또는 가스 점검 시점"

가치: 고장 → 요금 폭탄을 미리 막음. 사용자가 가장 고마워하는 기능.
```

#### ④ 탄소중립포인트 자동 연동

```
매월 사용량 → 전월 동기 대비 절감률 계산
5% 이상 절감 시:
  - 자동 알림: "이번 달 -7.2% 절감! 탄소중립포인트 1,400점 적립 가능"
  - 원클릭 신청 (한전 마이데이터 기반 자동 제출)
  - 연간 절약 + 포인트 누적 대시보드

추가:
  탄소 배출량 = kWh × 0.4434 (한국 전력 배출계수)
  "이번 달 절약: 38kWh = 탄소 16.8kg 감축 = 소나무 2.5그루 1년치"
```

### 7-3. 사용자 페르소나별 시나리오

#### 페르소나 A: 30대 맞벌이 부부 (3인 가구, 30평 아파트)

- 주요 고민: 여름 에어컨 요금
- 서비스 가치:
  - 6월부터 누진 진입 위험 사전 알림
  - 에어컨 패턴 분석 → "주말 낮 6시간 가동 vs. 오후 8시 이후 4시간 + 선풍기 병행 → 월 22,000원 차이"
  - 가전 ROI 계산 → "현재 7년 된 에어컨, 1등급 교체 시 연 8.4만원 절약, 정부 환급 30만원 감안 회수 4.2년"

#### 페르소나 B: 1인 가구 청년 (오피스텔)

- 주요 고민: 작은 집인데 누진 3단계 진입
- 서비스 가치:
  - 1인 가구에 누진제가 불리한 구조 자체를 시각화
  - "1인 가구임에도 4인 가구 평균 80% 사용. 대기전력이 주범"
  - 대기전력 차단 자동화 가전 추천
  - 비슷한 평형/가구원 수 이웃 평균과 비교 → 게임화

#### 페르소나 C: 노부부 (2인, 단독주택)

- 주요 고민: 보일러 + 전기 둘 다 부담
- 서비스 가치:
  - 음성 안내 + 큰 글자 모드
  - "이번 달 보일러 가동시간이 평소의 1.5배. 점검 권고"
  - 자녀에게 사용 패턴 공유 (효도 코칭)

#### 페르소나 D: 저소득층 (에너지바우처 수급자)

- 주요 고민: 바우처 잔액 관리
- 서비스 가치:
  - "이번 여름 바우처 잔액 4.2만원, 현재 속도라면 17일 소진"
  - 바우처 절약형 사용 패턴 자동 추천
  - 절약분을 동절기로 이월하는 옵션 안내

#### 페르소나 E: 전기차 보유자

- 주요 고민: 충전 비용
- 서비스 가치:
  - 경부하 시간대 자동 충전 알림
  - "오늘 새벽 1~5시 충전 시 동일 충전량 대비 38% 저렴"
  - V2G 시장 진입 시 자동 거래 게이트웨이 가능

#### 페르소나 F: 아파트 단지 관리사무소

- B2B2C 확장 가능성
- 단지 전체 평균 vs. 우리 단지 비교 대시보드
- 입주민 절약 챌린지 운영 도구
- 포스코이앤씨 "에너지 쉼표 챌린지" 사례 → 우리 서비스가 그 인프라가 됨

## 9. 비즈니스 모델 / 지속가능성

### 9-1. 공공 영역 (정책 제언)

이 서비스는 **공공 인프라로서 정부/한전이 직접 제공**하는 것이 가장 자연스럽다.

이유:
- 한전이 이미 데이터를 가지고 있음
- 마이데이터 인증 인프라도 있음
- 시민 모두에게 무료 보급 → 사회적 파급력 극대화

기획서의 핵심 정책 제언:
1. 한전 마이데이터 + NILM 기반 개인화 코칭을 **국가 표준 서비스**로 지정
2. 탄소중립포인트와 **자동 연동 의무화**
3. 취약계층 자동 가입 (동의 후) → 에너지 복지 기능
4. 데이터·AI 모델은 오픈 → 민간이 추가 부가서비스 개발 가능 (생태계)

### 9-2. 민간 영역 확장 가능성

| 모델 | 수익원 |
|------|------|
| B2C 프리미엄 | 고급 분석/예측은 유료 (월 3,000원 수준) |
| B2B 아파트 관리사무소 | 단지별 절약 챌린지 운영 도구 |
| B2B 보험사 | 가전 이상 감지 → 가전 보험 데이터 |
| B2B 가전 회사 | 교체 추천 채널 (LG, 삼성 효율 등급 가전 추천) |
| 데이터 인사이트 | 통계청·에너지공단에 익명 통계 제공 (가전 보급률 등) |
