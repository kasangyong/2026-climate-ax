## 1. 사회문제 해결

> 청정 시간은 단순한 에너지 앱이 아니다. 한국 에너지 시스템이 직면한 구조적 문제 6가지를 동시에 해결하는 공공 플랫폼이다.

---

### 1-1. AI 시대 전력 수요 폭증과 탄소중립 위협

#### 문제 현황

AI·클라우드 산업의 폭발적 성장은 전력 수요를 기하급수적으로 늘리고 있다.

| 지표 | 수치 | 출처 |
|---|---|---|
| 한국 데이터센터 전력 수요 | 2025년 4,461MW → 2028년 6,175MW (연 +11%) | 한국데이터센터연합회 |
| 2038년 예측 수요 | 2025년 대비 약 6배 | 산업부 전력수급기본계획 |
| 신규 전력 신청 집중도 | 67%가 수도권 | 한전 전력계통 현황 |

AI 서버는 24시간 일정한 대량 수요를 필요로 한다. 반면 재생에너지는 날씨·시간에 따라 발전량이 급변한다. 이 두 패턴이 충돌하면서 **화석연료 발전소가 변동성을 메우는 역설**이 발생한다.

```
【문제의 구조】

AI 데이터센터 수요  : ━━━━━━━━━━━━━━━━━━━━  (24시간 일정, 계속 증가)
재생에너지 발전량   : ▁▁▃▅████▅▃▁▁▃▅▃▁▁▁▁  (시간·날씨별 급변)

↓ 불일치 구간에서

LNG 화력발전 추가 가동 : ▁▁▃▅▁▁▁▁▃▅████▅▃▁  (탄소 배출 급증)
```

#### 현재 해법의 한계

- **공급 측 접근(발전소·ESS 증설)** : 1기당 수천억~수조원, 건설 기간 수년. 탄소중립 시한(2050) 내 전량 전환 불가.
- **수요 절약 캠페인** : "전기 덜 써라" → 시민 부담 크고 지속성 없음.
- **시간 분산형 수요반응** : 가장 효과적이나 시민 참여 도구가 없음.

#### 청정 시간의 해법

가정 부문 전력 사용의 **30~40%는 유연 부하** (세탁기·식기세척기·EV 충전·전기보일러 등)다. 이 유연 부하를 재생에너지가 남는 시간으로 이동시키면, 추가 발전소 없이 수요-공급 불일치를 완화할 수 있다.

```
청정 시간 시나리오 :
  오전 11시~오후 1시 → 태양광 정점, 재생E 남음
  → 청정 시간 알림 : "지금 세탁기·식기세척기 사용 권장"
  → 가정이 유연 부하를 이 시간으로 이동
  → 저녁 피크 수요 완화 → LNG 발전소 추가 가동 불필요
```

#### 기대 효과

| 지표 | 효과 | 산출 근거 |
|---|---|---|
| 피크 수요 감축 | -1GW (가정 5% 참여 시) | 가정 부문 피크 기여율 기반 추산 |
| LNG 화력 신규 건설 회피 | 약 5천억원 절감 | 500MW급 LNG 복합화력 건설비 기준 |
| CO₂ 감축 | 약 100만 톤/년 | 피크-오프피크 탄소강도 차이 × 이동 전력량 |

### 1-2. 재생에너지 출력제한 — 버려지는 전기

#### 문제 현황

재생에너지 보급이 빠르게 늘면서 전력망이 흡수하지 못하는 잉여 전기가 발생하고 있다. 이를 **출력제한(Curtailment)** 이라 하며, 발전소를 강제로 끄거나 발전량을 줄이는 조치다.

| 지표 | 수치 | 출처 |
|---|---|---|
| 2023년 태양광 출력제한량 | 약 700GWh 이상 | 전력거래소 |
| 2040년 예상 출력제한량 | 7.8~23.4 TWh | 제10차 전력수급기본계획 |
| 23.4 TWh의 의미 | 서울시 전체 1년 사용량의 약 1/3 | 서울시 연간 전력 소비 기준 |

```
【출력제한의 역설】

태양광·풍력이 충분히 생산하고 있는데
  → 전력망이 못 받아들임 (수요 부족)
  → 발전소를 강제로 끔
  → 발전 사업자 수익 손실
  → 재생에너지 투자 의욕 저하
  → 탄소중립 가속화에 역행
```

#### 현재 해법의 한계

- **대규모 ESS 설치** : 비용이 수조원 규모, 보급까지 수년 소요.
- **송전망 확충** : 주민 반대로 사업 지연, 건설에 10년 이상.
- **가정 수요 이동** : 가장 빠르고 저렴하지만, 시민이 알 방법이 없음.

#### 청정 시간의 해법

재생에너지가 남아서 버려지려는 시간에 가정 수요를 끌어당긴다. 출력제한이 예상되는 시간을 미리 청정 시간으로 지정하고, 가정이 세탁기·식기세척기·EV 충전을 그 시간으로 이동하면 잉여 전기가 소비되어 출력제한이 줄어든다.

```
출력제한 예방 시나리오 :
  [오전 11시] 태양광 과잉 발전 예측 → 출력제한 위기
  → 청정 시간 알림 : "지금 재생에너지가 남습니다. 가전 사용을 권장합니다"
  → 알림을 받은 가정들이 세탁기·식기세척기를 이 시간에 집중 가동
  → 잉여 전력 흡수 → 출력제한 불필요
```

#### 기대 효과

| 지표 | 효과 |
|---|---|
| 연간 출력제한 감소 | 약 -2TWh/년 (가정 5% 시간 이동 시) |
| 회수되는 재생E | 200만 가구 1년 사용량에 해당 |
| 재생에너지 사업자 수익 보호 | 버려지던 전기가 판매로 전환 |

### 1-3. 가정 DR 참여 정책 공백

#### 문제 현황

수요반응(DR, Demand Response)은 전력 수급 위기 시 가정·기업이 전기 사용을 줄이는 대신 보상을 받는 제도다. 한국에는 이미 이 제도가 운영되고 있지만, 가정 부문 참여가 극도로 저조하다.

```
【참여 격차 — 보상이 높은데 왜 안 쓰나】

AMI 보급 가구    ████████████████████████████████████████  2,005만 호
                 (잠재 DR 자원 전체 — 계량기 이미 설치 완료)

에너지캐시백     █████████████                              166만 호
                 절약형 DR | 보상 30~100원/kWh | "덜 써라"

국민DR           ▌                                           7,430호
                 시간이동형 | 보상 약 1,300원/kWh | 보상이 13~43배 높음

→ 보상이 13배 이상 높은 제도의 참여율이 1/200인 이유 :
   ① 알기 어렵다    — 국민DR이라는 제도를 아는 시민 극소수
   ② 참여하기 어렵다 — 전력거래소 별도 가입 + 발령마다 개별 신청
   ③ 정산이 어렵다  — 수동 확인 + 개별 요청
```

**국민DR과 에너지캐시백의 보상 단가 비교 :**

| 제도 | 보상 단가 | 참여 가구 | 연간 추산 보상 (1가구) |
|---|---|---|---|
| 에너지캐시백 | 30~100원/kWh | 166만 호 | 약 1~3만원 |
| 국민DR | 약 1,300원/kWh | 7,430호 | 약 5~15만원 |

**국민DR이 13~43배 높은 보상을 제공하는데도 참여율이 1/200에 그치는 것은 제도의 접근성 문제다.**

#### 현재 해법의 한계

- **홍보 강화** : 인지도는 올릴 수 있어도, 신청·참여의 복잡성 해소 불가.
- **보상 단가 상향** : 이미 1,300원/kWh로 충분히 높음. 단가 문제가 아님.
- **개별 가입 독려** : 근본적으로 수동 프로세스를 개선하지 않으면 한계.

#### 청정 시간의 해법

3단계로 장벽을 동시에 제거한다.

```
장벽 ① 알기 어렵다
  → 해법 : 매일 아침 "오늘 청정 시간 + DR 발령 가능성" 자동 알림
            "오후 2~5시 국민DR 발령 예정 — 5,200원 적립 가능"

장벽 ② 참여하기 어렵다
  → 해법 : 전력거래소 가입·신청 0회. 청정 시간 앱에서 1탭으로 자동 참여
            "[참여] 버튼 1개 → AMI 자동 측정 → 자동 정산"

장벽 ③ 정산이 어렵다
  → 해법 : AMI가 실제 감축량 자동 측정 → 다음달 전기요금 자동 차감
            별도 확인·신청 0회
```

#### 기대 효과

| 시나리오 | 참여 가구 | 효과 |
|---|---|---|
| 현재 (변화 없음) | 7,430호 | — |
| 청정 시간 도입 후 보수적 | 10만 호 | DR 자원 13배 확대 |
| 청정 시간 도입 후 목표 | 100만 호 이상 | 에너지캐시백 규모에 근접, 피크 -1GW 회피 |

### 1-4. 에너지 복지 사각지대

#### 문제 현황

에너지 취약계층은 이미 에너지 비용 부담이 크지만, 정부 지원 제도를 제대로 활용하지 못하는 경우가 많다.

| 지표 | 수치 | 출처 |
|---|---|---|
| 에너지바우처 수급 가구 | 약 130만 가구 | 한국에너지공단 |
| 주요 수급 대상 | 기초생활수급자 + 차상위 계층 중 노인·장애인·영유아 가구 | |
| 주된 어려움 | 잔액 소진 시점 파악 어려움, 디지털 신청 어려움 | |

**문제의 구조 :**
```
에너지바우처 수급자
  → 잔액이 얼마 남았는지 모름 (실시간 확인 어려움)
  → 추울 때 전기를 아끼다가 건강 피해
  → 에너지캐시백·국민DR 등 추가 보상 제도는 아예 모름
  → 디지털 기기 사용 어려워 온라인 신청 불가

결과 : 가장 지원이 필요한 계층이 가장 적은 혜택을 받는 역설
```

#### 현재 해법의 한계

- **복지관·주민센터 안내** : 고령·장애인 가구에게 오프라인 안내는 접근성 한계.
- **문자 발송** : 일방향 통보, 실제 행동 변화로 연결되지 않음.
- **디지털 리터러시 교육** : 시간이 오래 걸리고 효과 불확실.

#### 청정 시간의 해법

**Default Opt-In** 방식 : 에너지바우처 수급자가 청정 시간에 자동 가입되어 별도 신청 없이 보상을 받는다.

```
【취약계층 자동화 흐름】

에너지바우처 수급자 DB (에너지공단 API)
       ↓
청정 시간 자동 옵트인 (별도 신청 없음)
       ↓
    ┌──────────────────────────────────┐
    │  ① 바우처 잔액 실시간 알림        │
    │     "이번 달 남은 지원금 23,000원" │
    │  ② 에너지캐시백 자동 신청·적립    │
    │  ③ 청정 시간 알림형 참여           │
    │     (큰 글자 + 음성 안내 모드)     │
    │  ④ 자녀·가족 공유 알림            │
    └──────────────────────────────────┘
```

**취약계층 모드 UI 특징 :**
- 글자 크기 최대화 (노인 시력 고려)
- 음성 안내 지원
- 자녀·가족이 대신 설정·확인 가능한 공유 기능
- 알림은 단순하게 ("지금 세탁기 쓰면 돈 아낍니다")

#### 기대 효과

| 문제 | 해결 방식 | 기대 효과 |
|---|---|---|
| 바우처 잔액 모름 | 실시간 잔액 알림 | 바우처 미소진으로 인한 복지 손실 제거 |
| 캐시백 미신청 | 자동 옵트인 | 130만 가구 자동 보상 적립 |
| 디지털 접근성 부재 | 큰 글자·음성·가족 공유 | 디지털 격차 없이 에너지 복지 자동화 |

### 1-5. 시민의 환경 행동 동기 부족

#### 문제 현황

탄소중립은 한국 사회의 핵심 과제지만, 개인 수준의 행동 변화는 더디다. 이유는 단순하다 — **행동의 비용이 너무 크고, 보상이 너무 불분명하다.**

```
【기존 환경 행동 캠페인의 구조적 한계】

메시지  : "탄소중립을 위해 전기를 아껴주세요"
행동    : 더위 참기, 조명 끄기, 에어컨 자제
비용    : 즉각적이고 체감 가능한 불편
보상    : 추상적, 장기적, 개인 기여분 불가시
결과    : 지속 참여 어려움, 무임승차 문제

→ 행동 경제학 원칙 위반 :
  "즉각적 손실"이 "장기적 이익"보다 심리적으로 훨씬 크게 느껴진다
  (손실 회피 편향, Loss Aversion)
```

#### 현재 해법의 한계

- **도덕적 호소** : "미래 세대를 위해서"는 지속 동기 부여에 한계.
- **탄소중립포인트** : 좋은 제도지만 참여 방법이 분산되어 있고 피드백이 없음.
- **챌린지 캠페인** : 단발성 이벤트, 지속성 없음.

#### 청정 시간의 해법

행동 경제학 원칙을 역으로 활용한다. **즉각적 보상 + 구체적 가시화 + 게임화**로 환경 행동의 심리적 비용을 제거한다.

```
【청정 시간의 동기 설계】

행동    : 세탁기 시작 시간을 2시간 미룸 (불편 거의 없음)
보상    : 즉각적 — "탄소 -52%, 180원 절약, 포인트 +85점" (행동 직후)
가시화  : "소나무 2.1그루 1년치" (추상적 CO₂ kg을 체감 가능한 단위로 변환)
게임화  : "관악구 상위 18%" (사회 비교로 지속 동기 부여)
누적    : "올해 소나무 26그루, 보상 68,400원" (연간 성취감)
```

**행동 변화 설계 핵심 원칙 :**

| 원칙 | 기존 캠페인 | 청정 시간 |
|---|---|---|
| 즉각성 | 보상 수개월 후 | 행동 즉시 피드백 |
| 구체성 | "탄소중립에 기여" | "소나무 2.1그루 + 7,740원" |
| 부담 최소화 | 전기 사용 줄이기 | 사용 시간만 바꾸기 |
| 사회적 동기 | 없음 | 지역 순위, 친구 비교 |
| 자동화 | 매번 의식적 노력 | 자동 모드로 의식 불필요 |

#### 기대 효과

- 게임화·가시화를 통한 지속 참여율 향상 (영국 Octopus 참여자의 68%가 1년 후에도 시간 이동 습관 유지)
- "탄소중립이 돈이 된다"는 인식 전환 → 환경 행동의 대중화
- 지역 커뮤니티 단위 환경 경쟁 문화 형성

### 1-6. 가정 부문 에너지 정보 비대칭

#### 문제 현황

한국 시민은 전기를 쓸 때 그것이 어디서 왔는지, 탄소를 얼마나 배출하는지 전혀 알 수 없다. 이 정보 비대칭이 모든 문제의 근원이다.

```
【정보 비대칭의 구조】

알 수 있는 것 (시민) :
  ✅ 이번 달 전기 사용량 (kWh)
  ✅ 전기요금 (원)

알 수 없는 것 (시민) :
  ❌ 지금 이 순간 전기가 어디서 오는가 (발전 믹스)
  ❌ 지금 전기를 쓰면 탄소가 얼마나 나오는가 (탄소강도)
  ❌ 어떤 시간에 쓰면 탄소가 줄어드는가 (청정 시간)
  ❌ 내가 시간을 이동하면 얼마를 절감하는가 (절감 효과)

반면 해외에서는 :
  🇬🇧 영국 : 국가 전력망 실시간 탄소강도 공개 (carbon intensity API)
  🇺🇸 미국 : WattTime 5분 단위 한계 배출계수 상용 서비스
  🇪🇺 EU   : Electricity Maps 190개국 실시간 탄소강도 지도
```

**한국 탄소 배출계수 현황 :**

| 발표 형태 | 내용 | 한계 |
|---|---|---|
| 현재 공식 발표 | 연간 평균 0.4434 tCO₂/MWh (2023, 온실가스종합정보센터) | 시간대별 차이 반영 안 됨 |
| 필요한 형태 | 시간대별·계절별·지역별 탄소강도 실시간 공시 | 현재 없음 |
| 실제 격차 | 피크 시간 약 520~550 gCO₂/kWh vs 청정 시간 약 200~250 gCO₂/kWh | 2.7배 차이, 시민에게 미공개 |

#### 현재 해법의 한계

- **My한전 앱** : 총 사용량·요금만 제공. 탄소 정보 없음.
- **탄소중립포인트** : 절감량 집계는 하지만 시간대별 탄소 정보 미제공.
- **환경부 정보 공개** : 연평균만 발표, 실시간·시간대별 공시 없음.

#### 청정 시간의 해법

**한국 최초로 가정에 시간대별 탄소강도를 실시간 공개하고, 이를 행동 지침으로 연결한다.**

```
청정 시간 앱이 제공하는 정보 :
  ① 지금 이 순간 한국 전력망 탄소강도 : XXX gCO₂/kWh
  ② 24시간 탄소강도 예측 타임라인
  ③ "세탁기를 지금 돌리면 CO₂ XXg / 오후 7시에 돌리면 CO₂ YYg"
  ④ 내 가정의 오늘 탄소 배출 실시간 누적
  ⑤ 시간 이동으로 절감한 탄소 누적 (소나무 환산)
```

**정책 제언으로 연결 :**
청정 시간이 시범 서비스를 통해 시간대별 탄소강도 공시의 효과를 입증하면, 이를 근거로 정부의 **시간대별 탄소 배출계수 공시 의무화** 정책을 이끌어낼 수 있다. 이는 기업의 RE100·탄소중립 경영에도 직접 활용 가능하다.

#### 기대 효과

- 시민의 에너지 정보 접근권 실질화
- 시간대별 탄소강도 공시 의무화 정책 실현의 근거 마련
- 기업 RE100 달성을 위한 시간대별 재생에너지 매칭 인프라 선행 구축

### 2-3. 정책 제언 5종 (상세)

#### 정책 제언 ① — 국가 표준 가정 부문 청정 에너지 매개 플랫폼 지정

**배경** : 현재 한전·전력거래소·환경부·에너지공단이 각각 별도 제도를 운영하면서 시민 경험이 분산되어 있다. 공익 매개 플랫폼 하나가 이를 통합하면 모든 기관의 정책 효율이 올라간다.

**제언 내용** :
- 기후에너지환경부 + 한전 + 전력거래소 공동 운영 플랫폼으로 청정 시간 지정
- 운영 재원 : 분산에너지 시장 매출 일부 + 한전 캐시백 예산 자동화 절감분 + 국민DR 정산 환원
- 민간 운영사 위탁 또는 공공기관 직접 운영 병행 가능

**기대 효과** :
- 4개 제도 시민 접근성 통합 향상
- 정책 집행 비용 절감 (중복 홍보·신청 처리 비용)
- 가정 DR 참여율 0.45% → 5% 목표 달성

---

#### 정책 제언 ② — 시간대별 탄소 배출계수 공시 의무화

**배경** : 현재 한국은 연간 평균 배출계수만 공식 발표한다. 시간대별로 탄소강도가 2.7배 차이나는데도 시민은 이 정보에 접근하지 못한다.

**제언 내용** :
- 온실가스종합정보센터 + 전력거래소가 시간대별(30분 또는 1시간 단위) 탄소강도를 Open API로 실시간 공시
- 영국 National Grid ESO 모델 참조
- 기업 RE100 시간별 매칭 데이터로도 활용 가능하도록 표준화

**기대 효과** :
- 시민 탄소 인식 향상 → 자발적 시간 이동 유도
- 기업 탄소 경영 고도화 (시간별 재생에너지 조달 최적화)
- 청정 시간 앱의 알림 정확도 법적 근거 확보

---

#### 정책 제언 ③ — 국민DR 자동 참여 옵션 표준화

**배경** : 국민DR은 보상 단가가 높지만 수동 신청 구조로 인해 7,430호에서 정체 중이다.

**제언 내용** :
- 마이데이터 동의 사용자는 DR 발령 시 1탭으로 자동 참여 가능하도록 전력거래소 시스템 개선
- 청정 시간 같은 매개 서비스가 DR 참여를 대리 신청할 수 있는 위임 API 표준 마련
- AMI 자동 측정 → 자동 정산 파이프라인 법제화

**기대 효과** :
- 국민DR 가정 참여율 0.45% → 5% 이상 (100배 잠재력)
- 가정 DR이 실질적 계통 안정 자원으로 기능
- LNG 피크 발전소 1기 건설 회피 (약 5천억원 절감)

---

#### 정책 제언 ④ — 취약계층 에너지 서비스 자동 가입 (Default Opt-In)

**배경** : 에너지바우처 수급자 약 130만 가구가 복잡한 신청 절차로 인해 추가 지원 제도 혜택을 받지 못하고 있다.

**제언 내용** :
- 에너지바우처 수급자는 청정 시간(또는 이와 동등한 공인 플랫폼)에 자동 가입 (별도 신청 불필요)
- 바우처 잔액 실시간 알림 + 에너지캐시백 자동 신청·적립 자동화
- 디지털 취약계층을 위한 취약계층 모드(큰 글자·음성·가족 공유) 표준화

**기대 효과** :
- 130만 취약 가구 에너지 복지 자동화
- 디지털 격차로 인한 복지 사각지대 해소
- 에너지 빈곤 가구의 실질 에너지비 부담 경감

---

#### 정책 제언 ⑤ — 마이데이터 + 엣지 추론 표준 가이드 수립

**배경** : 전력 사용량 데이터는 가정 생활 패턴이 그대로 드러나는 민감한 정보다. 개인정보 보호 표준이 명확해야 마이데이터 생태계가 활성화된다.

**제언 내용** :
- 에너지 마이데이터 서비스에 차등 프라이버시(Differential Privacy) 적용 가이드 수립
- K-익명성 5 이상 의무화
- NILM 모델 단말 실행(엣지 추론) 옵션을 에너지 앱 심사 기준에 포함
- 원시 전력 데이터 서버 전송 없이 분석 결과만 전송하는 엣지 추론 표준 마련

**기대 효과** :
- 시민 개인정보 보호 강화 → 마이데이터 동의율 향상
- 에너지 데이터 활용 생태계 신뢰 기반 구축
- 글로벌 수준의 프라이버시 보호 표준 선도

### 2-4. 심사 기준별 대응 근거 (상세)

#### 데이터 융합성 (1순위 심사 기준)

청정 시간은 단일 데이터가 아닌 **6종 데이터의 실시간 융합**이 핵심이다.

| 데이터 | 역할 | 융합 방식 |
|---|---|---|
| 한전 AMI LP (15분) | 가정 전력 사용 패턴 | NILM으로 가전별 분리 → 유연 부하 식별 |
| EPSIS 발전 믹스 (실시간) | 탄소강도 실시간 산출 | 발전원별 비중 × 배출계수 가중 평균 |
| 온실가스종합정보센터 배출계수 | 탄소강도 환산 기준 | 발전원별 gCO₂/kWh 곱하기 |
| 기상청 동네예보 (일사·풍속) | 24h 탄소강도 예측 보조 | LSTM 모델 보조 입력 |
| 에너지공단 효율등급 DB | 가전 절감 효과 추산 | 절감량 × 효율등급 가중치 |
| 탄소중립포인트 | 실적 자동 적립 | API 연동 자동 집계 |

**6종 데이터가 만드는 최종 가치 :** 개별 데이터 하나만으로는 "지금 세탁기 돌리면 탄소가 얼마"를 계산할 수 없다. AMI(얼마나 쓰는가) + EPSIS(어디서 오는가) + 배출계수(탄소가 얼마인가) + 기상(앞으로 어떻게 될 것인가) + 효율등급(어떤 가전인가) 5종이 융합되어야 가구 맞춤 청정 시간 알림이 생성된다.

#### 창의성·독창성

| 독창성 항목 | 세계적 선례 | 한국 최초 여부 |
|---|---|---|
| 가정용 Carbon-Aware Computing | Microsoft Azure·Google Cloud (데이터센터 적용) | ✅ 한국 가정 최초 |
| NILM 기반 가전별 분석 상용화 | EDF 프랑스 (NILMFormer) | ✅ 한국 최초 |
| 4개 에너지 제도 자동 통합 | 없음 | ✅ 세계 최초 수준 |
| "덜 써라" → "다른 시간에 써라" | Octopus(영국), OhmConnect(미국) | ✅ 한국 최초 |
| 가정 DR 24시간 사전 알림 | 해외 일부 서비스 | ✅ 한국 최초 |

#### 논리적 타당성

모든 핵심 수치는 공식 출처 기반이다.

| 수치 | 출처 |
|---|---|
| 화석연료 발전 비중 49.5% (2025.4) | 전력거래소 EPSIS |
| 출력제한 예상 23.4 TWh (2040) | 제10차 전력수급기본계획 |
| 국민DR 참여 7,430호 | 전력거래소 공식 자료 |
| 에너지캐시백 166만 호 | 한전 공식 발표 (2025) |
| AMI 보급 2,005만 호 | 한국전력공사 공식 발표 (2024) |
| 탄소 배출계수 0.4434 tCO₂/MWh | 온실가스종합정보센터 (2023) |

| OhmConnect 200,000 가구 1GWh | OhmConnect 공식 발표 |

#### 실현 가능성

3~6개월 MVP가 가능한 이유 세 가지 :

```
① 법 개정 불필요
   분산에너지 활성화 특별법 2024.6 시행 완료
   마이데이터 관련 법제 기확립
   → 법률 리스크 없음

② 인프라·API 이미 존재
   AMI 2,005만 호 보급 완료
   한전 마이데이터 API 개방 완료
   EPSIS Open API 개방 완료
   NILMFormer 오픈소스 공개
   → 처음부터 만들 것이 없음

③ 해외 선례로 실현 가능성 검증
   Octopus Agile (영국) : 동일 개념 이미 수십만 가구 운영 중
   OhmConnect (미국) : 가정 DR 도매시장 자원화 이미 검증
   EDF/NILMFormer (프랑스) : NILM 상용화 이미 검증
   → 기술·사업 불확실성 최소화
```

#### 사회적 파급력

| 지표 | 규모 | 비교 기준 |
|---|---|---|

| CO₂ 감축 | 약 100만 톤/년 | 소나무 약 1억 4천만 그루 1년치 흡수량 |
| 가구당 경제 효과 | 연 5~10만원 자동 적립 | 에너지캐시백(연 1~3만원) 대비 3~10배 |
| 재생E 출력제한 감소 | -2TWh/년 | 200만 가구 1년 사용량 |

#### 신규 예산·법 개정 불필요

```
신규 법률 제정 : 0건
신규 인프라 설치 : 0건
신규 보상 제도 신설 : 0건

활용하는 것 :
  ✅ 이미 설치된 AMI 2,005만 호
  ✅ 이미 개방된 한전·EPSIS·환경부·기상청 API

  ✅ 이미 시행된 분산에너지 특별법
  ✅ 이미 공개된 NILMFormer 오픈소스
```
