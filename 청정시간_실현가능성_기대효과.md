## 1. 실현가능성

### 1-1. 즉시 실현 가능 조건 (법·제도·인프라)

**"법 개정 없이 지금 당장 시작할 수 있다"** 는 것이 핵심 강점이다.

| 조건 | 현황 | 활용 방법 |
|------|------|---------|
| **한전 AMI 인프라** | 2,005만 호 보급 완료 (2024년 기준, 한국전력공사 공식 발표) | 마이데이터 API로 15분 단위 LP 데이터 수집 |
| **한전 마이데이터 API** | 이미 개방됨 | 사용자 동의 1회로 AMI 데이터 연동 |
| **전력거래소 EPSIS API** | 이미 개방됨 (epsis.kpx.or.kr) | 실시간·24시간 예측 발전 믹스 데이터 활용 |
| **분산에너지 활성화 특별법** | 2023.6 제정 → 2024.6 시행 완료 (국가법령정보센터) | VPP 법적 근거, 가정 자원의 도매시장 참여 허용 |
| **탄소중립포인트 API** | 환경부 cpoint.or.kr 연동 가능 | 절감 실적 자동 집계 및 포인트 적립 연동 |
| **국민DR / 에너지쉼표 제도** | 이미 운영 중 (전력거래소) | 발령 신호 API 연동, 자동 참여 채널 구축 |
| **한전 에너지캐시백 제도** | 166만 호 가입 중 (2025년 기준, 한전 공식 발표) | 자동 신청·정산 경로 구축 |
| **에너지바우처 제도** | 운영 중 (한국에너지공단 energyv.or.kr) | 취약계층 잔액·소진 알림 자동화 |

> **핵심**: 신규 법 제정 0건, 신규 인프라 설치 0건. 기존 제도·API·인프라를 하나로 묶는 **통합 인터페이스**가 전부다.

> ⚠️ **단, 주의할 점**: "API 개방됨"과 "우리 서비스가 바로 사용 가능"은 다르다. 한전 마이데이터 사업자 협약·심사, 전력거래소 정산 연동 협약 등 **기관 협의 과정이 실제 병목**이다. 이 과정은 6~12개월이 소요될 수 있다.

### 1-2. 기술 실현 가능성

#### ① NILM (비침입 부하 모니터링) — 상용화 가능하나 한국 데이터 해상도 한계 주의

| 근거 | 내용 |
|------|------|
| **NILMFormer (KDD 2025)** | Transformer 기반 SOTA, EDF 프랑스가 실서비스 백본으로 채택. 논문 arXiv:2506.05880 |
| **오픈소스 완비** | NILMTK, Torch-NILM, NeuralNILM-PyTorch 등 즉시 활용 가능 |
| **공개 NILM 데이터셋** | REDD(미국), UK-DALE(영국), ECO(스위스) — 사전학습 재료 |
| **Active Learning 활용** | 소수 라벨 데이터로 미세조정 가능 → 라벨링 비용 대폭 절감 |

> ⚠️ **중요한 기술 제약 — 한국 AMI 데이터 해상도 문제**
>
> NILM 모델의 성능은 데이터 해상도에 강하게 의존한다.
>
> | 데이터 | 해상도 |
> |---|---|
> | UK-DALE, REDD (공개 학습 데이터) | 1초~6초 단위 |
> | 프랑스 EDF 실서비스 데이터 | 수 분 이내 (정확한 해상도 비공개) |
> | **한국 AMI LP 데이터** | **15분(900초) 단위** |
>
> 15분 단위 데이터는 학습 데이터 대비 해상도가 150배 낮다. 이 상태에서는 **에어컨·전기보일러처럼 전력이 뚜렷하게 큰 대형 가전의 온·오프 여부 정도만 구분 가능**하며, 세탁기·식기세척기처럼 전력 규모가 유사한 가전의 구별은 현 데이터로는 어렵다.
>
> → **MVP 전략**: 대형 가전(에어컨·전기차 충전·전기보일러) 위주의 패턴 분석에 집중하고, 세밀한 가전 분리는 **한국 시범 가구 LP 1분 단위 데이터 개방 이후 단계적으로 고도화**한다.
>
> → **정책 제언으로 연결**: AMI LP 15분 → 1분 단위 개방이 NILM 고도화의 전제 조건임을 정책 제언에 명시. 이 자체가 공모전 제안의 강점이 된다.

#### ② 탄소강도 예측 — 한국 자체 산출 가능

- EPSIS 시간대별 발전 데이터 + 온실가스종합정보센터 발전원별 배출계수 → **시간별 탄소강도(gCO₂/kWh) 자체 산출**
- 글로벌에서는 WattTime(5분 단위, docs.watttime.org), Electricity Maps(190개국, electricitymaps.com)가 이미 표준화
- 한국 배출계수 기준: 전력 평균 0.4434 tCO₂/MWh (온실가스종합정보센터 2023년 국가 배출계수)
- 피크 시간대(18~21시)와 오프피크 시간대(태양광 최대, 10~15시)의 탄소강도 차이는 **약 100~200 gCO₂/kWh** 수준으로 추정됨

#### ③ 기술 스택 — 검증된 표준 기술만 사용

```
[AI 모델]
  - NILM: NILMFormer 한국 가정 미세조정 (대형 가전 위주, 단계적 확대)
  - 탄소강도 예측: LSTM/Transformer + 발전 믹스 시계열
  - 패턴 분류: K-means 클러스터링

[서빙 인프라]
  - FastAPI + ONNX Runtime (실시간 추론)
  - Firebase Cloud Messaging / APNs (푸시 알림)
  - PostgreSQL + TimescaleDB + Redis

[프라이버시]
  - 마이데이터 표준 인증 (사용자 동의·철회)
  - 엣지 추론, 차등 프라이버시, K-익명성 5 이상
```

### 1-3. 단계별 확장 로드맵

```
[Phase 1] MVP 구축 — 3~6개월
  ✅ EPSIS 탄소강도 산출 엔진 (API 즉시 사용 가능)
  ✅ 청정 시간 알림 (푸시)
  ✅ 기존 4대 제도 자동 연동 UI (실 API 연동은 기관 협약 진행 중)
  - 한전 마이데이터: 사업자 협약 신청 → 승인까지 병행 진행
  → 목표: 시범 1,000가구 파일럿

[Phase 2] NILM 특화 — 6~12개월
  - 시범 가구 100~500호 가전 라벨링
  - 한국 가정 NILMFormer 미세조정 (대형 가전 위주)
  - 가전별 개인화 알림 (에어컨·EV·전기보일러 등)
  - 자동 모드 (스마트가전 API 연동)
  → 목표: 1만 가구 확대

[Phase 3] 지역·단지 확장 — 12~24개월
  - 아파트 단지 B2B (포스코이앤씨 등 협력)
  - 지자체 분산에너지 특구 시범
  - 지역별 VPP 도매시장 참여
  → 목표: 10만 가구 / 도매시장 자원화

[Phase 4] 국가 표준화 — 24개월 이후
  - 한전·전력거래소 공식 채택
  - 에너지바우처 수급자 Default Opt-In
  - 국가 표준 가정 부문 청정 에너지 매개 플랫폼 지정
  → 잠재 도달: 2,005만 가구
```

## 2-1. 사용자(가정) 측면

> **"전기를 덜 써라"가 아니라 "같은 양을 더 깨끗한 시간에 써라" — 생활 편의를 유지하면서 돈과 탄소를 동시에 아낀다.**

### 경제적 보상

기존에는 에너지캐시백·국민DR·탄소중립포인트·에너지바우처를 각각 따로 신청·관리해야 했다. 청정 시간은 4개를 앱 하나에서 자동 통합한다.

| 보상 종류 | 단가 | 연간 예상 | 신뢰도 | 계산 근거 |
|---------|------|---------|--------|---------|
| **한전 에너지캐시백** | 30~100원/kWh | **가구당 약 31,400원** | ⭐⭐⭐⭐⭐ | 실측: 522억원 ÷ 166만 호 = 31,445원. 출처: 한전 공식 발표, 인사이트코리아 2025.3 |
| **국민DR 보상** | **1,300원/kWh** | 연 10~30회 발령 × 가구당 0.5~1kWh 감축 → **약 7,000~39,000원** | ⭐⭐⭐⭐ | 단가: 전력거래소 공식 단가. 발령 빈도 10~30회: 전력거래소 과거 자료 기반 추정 |
| **탄소중립포인트** | 절감 비율 기반 | **수천~1만원** | ⭐⭐⭐ | 환경부 cpoint.or.kr 기준. 5% 이상 절감 가구 대상 |
| **에너지바우처 자동 연계** | 수급자 전용 | 복지 접근성 향상 | ⭐⭐⭐⭐ | 현재 수동 신청 → 자동 옵트인으로 전환 |

**가구 유형별 연간 합산 (캐시백 + DR + 포인트)**

| 가구 유형 | 연간 보상 | 신뢰도 |
|---|---|---|
| 일반 가정 (EV·TOU 없음) | **연 3~6만원** | ⭐⭐⭐⭐ |
| TOU 요금제 + 적극 참여 | **연 최대 20~40만원** | ⭐⭐⭐ |
| EV 보유 가구 (충전 최적화 포함) | **연 20만원 이상** | ⭐⭐⭐⭐⭐ |

> 국민DR 단가(1,300원/kWh)는 에너지캐시백(30~100원/kWh)보다 **13~43배 높다.** 이 차이를 몰라서, 신청이 복잡해서 참여 못 한 가구가 전국에 2,000만 호 가까이 있다.

### 맞벌이 가구 시뮬레이션 (30대, 3인, 스마트 가전 보유)

```
[기존 패턴]
  저녁 7시 세탁기(0.5kWh) + 식기세척기(1kWh) 동시 가동
  → LNG 피크 시간대, 누진 2단계 경계

[청정 시간 적용]
  새벽 2시 자동 예약 (풍력·원자력 중심, 탄소강도 낮음)

  절감 항목별 계산:
    ① 캐시백: 연간 약 12,000~15,000원 (3% 이상 절감 가구 기준)
    ② 국민DR 연 15회 × 1kWh × 1,300원 = 19,500원
    ③ 누진제 절약: 가전 야간 이동 → 월 10kWh 절감
       10kWh × (182.6 - 108.6)원 = 740원/월 → 연 약 8,900원
       (TOU 요금제 적용 시 월 5,000~20,000원 추가)

  현실적 합산:
    일반 가정 (TOU 미적용): 연 약 40,000~80,000원
    TOU 요금제 + 적극 참여: 연 최대 20~40만원
```

### EV 보유 가구 시뮬레이션

```
[조건]
  연간 주행: 15,000km
  전비: 6km/kWh (한국에너지공단 2025 전기차 평균 전비 기준)
  연 필요 충전량: 15,000 ÷ 6 = 2,500kWh

[경제 절감]
  경부하 vs 최대부하 단가 차이: 약 70~80원/kWh (한전 주택용 전기요금 기준)
  2,500kWh × 72원 = 180,000원 → 연 약 18만원

[탄소 감축]
  피크 탄소강도: 약 500 gCO₂/kWh / 오프피크: 약 400 gCO₂/kWh
  차이: 약 96 gCO₂/kWh
  2,500kWh × 96 gCO₂/kWh = 240kg CO₂ 감축

결론: EV 충전 최적화만으로 연 18만원 절감 + CO₂ -240kg  ⭐⭐⭐⭐⭐
```

### 취약계층 에너지 복지 자동화

- 에너지바우처 수급자 자동 가입(Default Opt-In) → 신청 없이 보상 수령
- 잔액 자동 알림 + 예상 소진 시점 사전 안내
- 큰 글자 모드·음성 안내·보호자 알림 공유 → 디지털 격차 해소
- 저효율 가전 교체 환급 정보 자동 안내

| 효과 항목 | 내용 |
|---------|------|
| **소나무 환산** | 소나무 1그루 연 CO₂ 흡수량 약 6.6kg (국립산림과학원 기준). 가구당 월 10kg 감축 = 소나무 약 1.5그루/월 |
| **주간 리포트** | "이번 주 청정 시간 참여 12회 → CO₂ -14.3kg = 소나무 2.1그루 1년치, 보상 7,740원" |
| **게임화** | 지역 순위·월간 챌린지·친구 비교로 지속 참여 유도 |

## 2-2. 정부(기후에너지환경부·산업통상자원부) 측면

> **"새 발전소 짓지 않고 탄소 줄이는 가장 빠른 방법 — 가정 수요를 분산하면 LNG 1기가 필요 없어진다."**

### 전력망 안정화 — 피크 수요 감축

| 지표 | 수치 | 신뢰도 | 근거 |
|---|---|---|---|
| **피크 수요 감축 (5% 참여, 약 100만 호)** | **-0.5~0.7GW** | ⭐⭐⭐ | 한국 가정 부문 피크 약 10~12GW (전체 피크 80~85GW 중 가정 약 14%, 한전 전력통계) × 5% × 유연 부하 비율·동시성 계수 적용 |
| **피크 수요 감축 (10% 참여, 장기 목표)** | **-1GW** | ⭐⭐⭐ | 위 동일 방식, 참여율 배증 |
| **LNG 화력발전 건설 회피 비용** | **5,000~7,500억원** | ⭐⭐⭐⭐ | 500MW급 LNG 복합화력 건설 단가 kW당 100~150만원 (에너지경제연구원·한전 참고) |
| **가정 DR 자원의 VPP 등록** | 법적 근거 완비 | ⭐⭐⭐⭐⭐ | 분산에너지 활성화 특별법 2024.6 시행. 가정 자원 도매시장 참여 허용 |

---

### CO₂ 감축 — 직접 + 간접 구분 계산

**직접 효과: 탄소강도 차이**

```
참여 가구: 100만 호 (2,005만 × 5%)
가구당 연간 이동 전력: 약 150kWh (세탁기·식기세척기 등)

피크 (18~21시) 탄소강도:   약 500~550 gCO₂/kWh (LNG·석탄 집중)
오프피크 (10~15시) 탄소강도: 약 300~400 gCO₂/kWh (태양광 피크)
차이: 평균 약 150 gCO₂/kWh
근거: EPSIS 발전 믹스 + 온실가스종합정보센터 발전원별 배출계수

직접 감축: 100만 호 × 150kWh × 150 gCO₂/kWh ≈ 15~25만 tCO₂
```

**간접 효과: 재생에너지 출력제한 감소**

```
출력제한 감소량: -1~2TWh (가정 수요가 재생E 과잉 시간에 집중 배치 시)
출력제한 1TWh 감소 = 화석연료 발전 1TWh 대체
한국 평균 배출계수 0.4434 tCO₂/MWh (온실가스종합정보센터 2023)
1,000,000 MWh × 0.4434 ≈ 44만 tCO₂

간접 감축: 44~88만 tCO₂
```

**종합**

| 구분 | 감축량 | 신뢰도 |
|---|---|---|
| 직접 (탄소강도 차이) | **15~25만 tCO₂/년** | ⭐⭐⭐ |
| 간접 (출력제한 흡수) | **44~88만 tCO₂/년** | ⭐⭐⭐ |
| **합계** | **59~113만 tCO₂/년** | ⭐⭐⭐ |

> **발표 권장**: "약 20~60만 톤/년 (직접 효과 기준)" 메인 제시. "출력제한 흡수 포함 시 최대 100만 톤"은 잠재 상한으로 부기.

## 2-3. 한전·전력거래소 측면

> **"가정 DR 자원을 VPP로 묶으면 LNG 예비력 비용이 줄고, 2,005만 호 AMI 투자의 실질 수익이 생긴다."**

### 피크 관리 비용 절감

- 한국 전력 도매시장에서 피크 시간대 SMP(계통 한계가격)는 오프피크 대비 급등하는 구조
- 가정 DR이 피크를 -0.5~1GW 감축하면 **SMP 급등 구간 완화 → 전력 구매 비용 절감**
- 현재 피크 대응용 예비력 유지 비용(용량 요금) 절감 효과

### AMI 2,005만 호 투자 수익 실현

- 한전이 2,005만 가구에 설치한 AMI는 지금까지 단방향 검침 용도가 주
- 청정 시간 연동 시 AMI 데이터가 **실시간 수요반응 자원으로 전환** → 투자 대비 활용도 급상승
- AMI LP 데이터 개방 확대 시 정부 국가중점데이터 활용 실적 확보

### 가정 부문 VPP 자원 확보

- 분산에너지 활성화 특별법(2024.6) 시행으로 가정 수요반응 자원의 도매시장 참여가 법적으로 허용됨
- 청정 시간이 중간 집계자(Aggregator) 역할 → 한전·거래소 입장에서 산업용 DR 외 **가정용 DR 포트폴리오 다변화**
- 현재 가정 DR 자원 거의 0 → 청정 시간 10만 호 참여 시 약 100~200MW 규모 가상발전소 형성 가능

| 지표 | 수치 | 신뢰도 |
|---|---|---|
| 잠재 VPP 규모 (10만 호 × 가구당 1~2kW 유연 부하) | **100~200MW** | ⭐⭐⭐ |
| 피크 SMP 절감 (간접 추정) | 피크 구간 1GW 감축 시 SMP 5~10% 완화 | ⭐⭐ |

## 2-4. 재생에너지 발전사업자 측면

> **"출력제한 때 버려지는 전기가 수익이다. 가정 수요를 끌어오면 출력제한이 줄고 태양광·풍력 투자 회수 기간이 단축된다."**

### 출력제한 감소 → 수익 보호

- 현재 출력제한 발생 시 태양광·풍력 사업자는 **발전량에 대한 보상 없이 발전을 중단**해야 함
- 2040년 재생에너지 비중 35% 시 연 출력제한 23.4TWh 예상 (에너지경제연구원 2025)
- 청정 시간이 재생E 과잉 시간대에 가정 수요를 유도하면 출력제한 -1~2TWh 직접 감소

| 지표 | 수치 | 신뢰도 | 근거 |
|---|---|---|---|
| 출력제한 감소 | **-1~2TWh/년** | ⭐⭐⭐ | 가정 수요 이동이 태양광·풍력 과잉 시간에 집중 배치될 경우 |

### 재생에너지 투자 불확실성 감소

- 출력제한 위험이 줄어들면 태양광·풍력 사업의 수익 예측 가능성 향상
- 재생에너지 추가 설치 시 계통 수용성 문제가 완화되어 **신규 재생에너지 투자 촉진**
- 선순환 구조: 가정 수요 유연화 → 출력제한 감소 → 재생E 투자 확대 → 재생E 비중 증가 → 탄소강도 하락

## 2-5. 가전·EV 산업계 측면

> **"스마트 가전이 실제로 '스마트'해지는 순간 — 청정 시간 연동이 스마트가전의 킬러 기능이 된다."**

### 스마트 가전 구매 유인 증가

- 현재 스마트 세탁기·식기세척기 구매자의 상당수가 스마트 기능을 활용하지 않음
- 청정 시간 연동 시 "빨래만 넣어두면 새벽에 자동으로 가장 싼·깨끗한 시간에 돌아간다" → **스마트 가전의 실질적 편익 체감**
- SmartThings(삼성)·ThinQ(LG) API 활용 확대 → 플랫폼 생태계 강화

| 가전 종류 | 청정 시간 연동 시 기대 효과 |
|---|---|
| 스마트 세탁기·식기세척기 (SmartThings·ThinQ API) | 예약 자동화 실현 → 스마트 기능 체감 → 구매 유인 강화 |
| EV 충전기 (약 50만 대 보급, 2024년 기준) | 충전 시간 최적화 → EV 운영비 절감 마케팅 소재 |

### EV 제조사·충전 사업자

- 현대·기아 차주 앱(현대 MyHyundai, 기아 Connected) 연동 시 **충전 비용 절감 기능**으로 차별화
- 충전 사업자(차지비·환경부 충전인프라 등): 청정 시간 연동으로 수요 분산 → 충전기 과부하 완화
- EV 보유 가구 충전 최적화 연 **18만원 절감** = EV 구매·운영 비용 절감 마케팅 소재로 활용 가능

## 3. 종합 판단

### 실현가능성 종합

```
즉시 가능 (0~6개월):
  ✅ 법 개정 불필요
  ✅ 추가 인프라 불필요 (AMI 2,005만 호 기완료)
  ✅ EPSIS·기상청 API 개방
  ✅ 오픈소스 AI 모델 완비 (NILMFormer 등)
  ✅ 분산에너지 특별법 시행 완료

병행 진행 필요 (0~12개월):
  → 한전 마이데이터 사업자 협약 (최대 병목)
  → 전력거래소 국민DR 자동 정산 협약
  → 환경부 탄소중립포인트 연동 협약

단계적 강화 (6~24개월):
  → NILM 한국 가정 특화 학습 (AMI 1분 데이터 개방 선결 조건)
  → 아파트 단지·지자체 시범
  → 국가 표준 플랫폼 지정
```

### 기대효과 핵심 3줄 (수정본)

> 1. **계통 효과**: 가정 5% 참여(100만 호) 시 피크 **-0.5~0.7GW**, 장기 10% 참여 시 -1GW = LNG 화력 1기 건설 회피 (건설비 5,000~7,500억원 절약)
> 2. **환경 효과**: CO₂ 직접 감축 약 **15~25만 톤/년** + 재생에너지 출력제한 흡수 포함 시 최대 **100만 톤/년** (직접+간접 합산)
> 3. **시민 효과**: 가구당 연 **3~6만원** 자동 보상 (EV 보유 시 연 20~40만원), 국민DR 참여 **7,430호 → 10만 호 이상** 확대
