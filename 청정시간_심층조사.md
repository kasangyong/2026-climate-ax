# 청정 시간 (Green Hour) — 심층 조사 노트

> 가정 부문 AI 수요반응(DR) 매칭 플랫폼
> 2026 기후부 AX 아이디어 경진대회 (사업·정책 아이디어 기획 / 에너지효율·에너지환경 미션)
> 기준일: 2026.05.10

---

## 0. 한 줄 요약

> **"내가 새벽 2시에 세탁기를 돌리면 한국 전체 탄소 배출이 줄어든다. AI가 그 시간을 알려주고, 자동으로 보상까지 챙겨준다."**

가정마다 흩어진 AMI 데이터·재생에너지 발전 신호·시간대별 탄소 배출계수를 AI로 융합해, 각 가정에 **"지금 가전을 돌리면 가장 친환경적인 시간"** 을 선제적으로 알려주는 공공 플랫폼. 참여 시 한전 에너지캐시백·국민DR 보상·탄소중립포인트가 자동 적립된다.

---

## 1. 왜 이 주제인가 — 두 개의 큰 흐름이 충돌하고 있다

### 흐름 1: AI 시대의 전력 수요 폭증

- 한국 데이터센터 전력 수요: **2025년 4,461MW → 2028년 6,175MW** (연평균 +11%)
- 2038년 예측: 2025년 대비 **약 6배** 증가
- 2024.8~2025.6, 11개월간 신규 전력 신청 290건 중 **195건(67%)이 수도권 집중**

### 흐름 2: 재생에너지 비중 급증과 변동성

- **2025년 4월, 한국 화석연료 발전 비중이 49.5%로 사상 처음 50% 아래로 떨어짐**
- 태양광 발전 비중 9.2%로 사상 최고치 갱신
- 2040년 재생에너지 비중 30% 가정 시 **출력제한량 7.8TWh** 발생 예상
- 35%로 가면 **23.4TWh 버려짐** (서울시 전력 사용량의 1/3 규모)

### 충돌의 본질

```
AI 데이터센터 = 24시간 일정한 대량 수요
재생에너지   = 시간대별·날씨별 변동 공급
                 ↓
공급-수요 불일치 → 화석연료 발전소가 변동성을 메움
                 → "재생에너지 늘려도 탄소 안 줄어드는" 역설
                 → 출력제한으로 깨끗한 전기 버려짐
```

해법은 두 가지:
1. **공급 측**: 발전소 더 짓기, ESS 더 깔기 → **막대한 비용·시간**
2. **수요 측**: 시간을 분산해 변동성을 흡수 → **거의 무료, 즉시 가능**

> 가정 부문 수요 분산이 답이다. 그런데 **가정에 그걸 알려주는 도구가 한국에 없다.**

## 2. 한국이 처한 상황 — 충격적인 수치들

### 2-1. 가정 부문 = 거대한 잠재력

| 지표 | 수치 | 의미 |
|------|------|------|
| 한전 AMI 보급 | **2,005만 호 (2024 완료)** | 인프라는 이미 구축됨 |
| 한전 에너지캐시백 참여 | **166만 호 (2025)** | 시민 참여 의지 입증 |
| 캐시백으로 절감한 전력 | **337GWh (2024)** | 충주시 전체 가정 1년 사용량 |
| 캐시백 환급액 | **522억원** | 가구당 연 49,000원 |
| 캐시백 CO₂ 감축 | **15만 톤** | 소나무 2,300만 그루 효과 |

### 2-2. 그러나 — 가정 DR 참여는 처참하다

한전 에너지캐시백(절약형)은 166만 가구 참여. 그런데 **시간대 분산형 수요반응(DR)은 거의 무인지대**다.

| 제도 | 참여 가구 수 | 보상 단가 |
|------|-----------|---------|
| 한전 에너지캐시백 (월간 절약형) | **166만 호** | 30~100원/kWh |
| 국민DR / 에너지쉼표 (시간 분산형) | **7,430호** ← 캐시백의 0.45% | **약 1,300원/kWh** |

> 보상 단가가 13~43배 더 높은데도 참여율은 0.45%. 이유: **알기 어렵고, 참여하기 어렵고, 효과를 모른다.**

이게 핵심 정책 공백이다. AMI 인프라 + 시민 참여 의지는 있는데, **시민-DR을 매개하는 똑똑한 인터페이스가 없다.**

### 2-3. 시간대별 탄소 격차는 크다

같은 1kWh를 써도 시간에 따라 탄소 배출이 달라진다:
- **태양광 피크 시간 (오전 10~오후 3시)**: 한 kWh당 탄소 적음
- **저녁 피크 시간 (오후 6~9시)**: LNG·석탄 대거 가동 → 탄소 폭증
- **새벽 풍력 강풍대**: 풍력 발전 폭주 → 깨끗한 전기 남음

WattTime, Electricity Maps 같은 글로벌 서비스는 이 정보를 5분 단위로 제공한다. **한국은 EPSIS 데이터로 산출 가능하지만, 시민용 서비스는 없다.**

### 2-4. 결론적 진단

```
한국이 가진 것:
  ✅ AMI 2,005만 호
  ✅ EPSIS 시간대별 발전 데이터
  ✅ 분산에너지 활성화 특별법 (2024.6 시행)
  ✅ 한전 에너지캐시백 166만 가구 참여
  ✅ 국민DR 제도 (보상 단가 매우 높음)
  ✅ 탄소중립포인트 제도

한국이 가지지 못한 것:
  ❌ 이 모든 것을 시민 입장에서 묶어주는 단일 인터페이스
  ❌ "지금 무엇을 하면 환경에 좋고 보상받는지" AI 알림
  ❌ 자동화된 가정 부문 DR 참여 채널
```

## 3. 핵심 개념 — Carbon-Aware Computing for Households

### 3-1. 개념 출발점

데이터센터 업계는 이미 **Carbon-Aware Computing**을 활용한다:
- Microsoft Azure: Carbon-Aware SDK (오픈소스)
- Google: 작업부하를 탄소 낮은 지역·시간으로 분산
- 학계: Carbon-Aware Kubernetes 등

> 데이터센터에서 작동하는 원리를 **가정에 적용**하는 것이 이 아이디어의 핵심.

### 3-2. 가정용 Carbon-Aware의 정의

```
[전제] 모든 가전 사용 = 시간대별로 다른 탄소 배출

[행동 변화] AI가 가정 사용 패턴을 학습한 뒤,
            "지금 이 가전을 돌리면 가장 깨끗하다"는 시간을 알림

[보상] 시간 변경에 참여한 가구에 탄소 감축량 + 요금 절감액 + 포인트 자동 정산
```

### 3-3. 작동 원리 도식

```
┌───────────────────────────────────────────────────────┐
│                  데이터 입력 레이어                       │
├───────────────────────────────────────────────────────┤
│ • 한전 AMI LP (15분 단위, 우리 집 사용 패턴)              │
│ • EPSIS 발전 믹스 (실시간 + 24시간 예측)                  │
│ • 온실가스 시간별 배출계수                                 │
│ • 기상청 예보 (태양광·풍력 발전량 예측 변수)              │
│ • SMP·계통 한계 데이터 (출력제한 위험 시간)               │
└───────────────────────────────────────────────────────┘
                         ↓
┌───────────────────────────────────────────────────────┐
│                  AI 분석 레이어                          │
├───────────────────────────────────────────────────────┤
│ • NILM: 가정의 가전별 사용 시간·전력 분리                 │
│ • 24시간 시간별 탄소강도(gCO₂/kWh) 예측                  │
│ • 가정별 "유연한 가전" 식별 (세탁기, 식기세척기, EV 등)   │
│ • 최적 시간 매칭: 청정 시간 (Green) / 회피 시간 (Red)    │
└───────────────────────────────────────────────────────┘
                         ↓
┌───────────────────────────────────────────────────────┐
│                  사용자 인터페이스                        │
├───────────────────────────────────────────────────────┤
│ ① 청정 시간 알림: "지금 풍력 폭주 → 세탁기 OK"             │
│ ② 회피 알림: "오후 6~9시 LNG 가동 → 식기세척기 늦추기"     │
│ ③ 자동 모드: 스마트 가전 연결 시 자동 시간 이동            │
│ ④ 누적 가시화: "이번 주 절감 = 소나무 4.2그루 1년치"      │
└───────────────────────────────────────────────────────┘
                         ↓
┌───────────────────────────────────────────────────────┐
│                  자동 보상 정산                          │
├───────────────────────────────────────────────────────┤
│ • 한전 에너지캐시백 자동 신청·정산                        │
│ • 국민DR 발령 시 자동 참여 → 1,300원/kWh 보상            │
│ • 탄소중립포인트 자동 적립 (5% 절감 시)                   │
│ • 시간대별 요금제(TOU) 적용 시 절약액 가시화               │
└───────────────────────────────────────────────────────┘
```

### 3-4. 핵심 차별화: "절약" vs "이동"

| 구분 | 한전 에너지캐시백 | **청정 시간** |
|------|--------------|------------|
| 행동 요구 | 총량 줄이기 | **시간 옮기기** (총량 유지 가능) |
| 사용자 부담 | 더 더위 참기, 덜 쓰기 | **편의는 그대로, 시간만 이동** |
| 환경 효과 | 탄소 감축 | 탄소 감축 + **재생에너지 통합** |
| 정책 효과 | 절약 | **계통 안정화 + 출력제한 감소** |

> 이 차이가 결정적이다. **"덜 써라"는 부담이 크지만, "다른 시간에 써라"는 부담이 작다.** 그래서 참여율을 폭발적으로 늘릴 수 있다.

## 4. 해외 검증 사례 — 이미 시장이 작동한다

### 4-1. Octopus Agile (영국) — 가장 강력한 벤치마크

```
서비스 구조:
  - 30분 단위 동적 요금 (도매가 연동)
  - 매일 다음날 24시간 요금 사전 공개
  - 사용자가 가전을 시간 이동 → 자동 절약
  - "Plunge Pricing": 음(-)의 요금 발생 시 사용자에게 돈을 지불
  
결과:
  - 영국 시민이 자발적으로 가전 시간 이동
  - 탄소 감축 + 요금 절감 동시 달성
  - 시간 이동만으로 가구당 연 30~50% 요금 절감 가능
```

> **시민이 시간 이동을 하더라**라는 것이 이미 영국에서 입증됐다.

### 4-2. OhmConnect (미국) — 가정 DR 시장 검증

```
캘리포니아·텍사스 운영 결과:
  - 약 200,000 가구 참여
  - 2020년 8월 한 주에 1GWh 감축 → 100만 달러 보상
  - 텍사스 폭염: 65MWh 감축, 가구당 연 50~100kWh 시간 이동
  - 가구당 월 5~7달러 + 변동 보상

비즈니스 모델:
  - 각 가정의 시간 이동을 "가상발전소(VPP)"로 묶어 도매시장 매도
  - 매출의 일부를 가정에 PayPal/Venmo로 지급
  - 게임화: 절감 챌린지, 친구 비교
```

> **가정 부문 DR이 도매시장에서 자원으로 거래된다**는 사실을 OhmConnect가 증명했다. 한국은 분산에너지 특별법으로 이 길이 열렸지만 가정 매개 서비스가 없다.

### 4-3. EDF (프랑스 전력공사) — NILMFormer 실서비스

- KDD 2025에서 발표된 SOTA NILM 모델 [NILMFormer](https://arxiv.org/abs/2506.05880)
- EDF가 **자사 모니터링 서비스 백본으로 채택, 수백만 가구 가전별 분석 제공 중**
- 한국 한전은 같은 데이터·기술력 보유했지만 안 함

### 4-4. WattTime / Electricity Maps — 탄소 강도 데이터

```
WattTime API:
  - 5분 단위 한계 배출계수(MOER, lbs CO₂/MWh)
  - "이 1kWh가 만드는 추가 탄소" 정확히 계산
  - Microsoft Azure, Google Cloud 모두 채택

Electricity Maps:
  - 190개국 평균 탄소 강도 데이터
  - 24시간 예측 + 실시간 + 과거 통합
  - 한국 데이터도 제공 (정확도는 한전 EPSIS가 더 높음)
```

> 한국에서는 EPSIS의 시간대별 발전 데이터 + 발전원별 배출계수로 **자체 산출 가능**. 단지 시민용으로 가공된 적이 없을 뿐.

### 4-5. 사례 종합

```
영국 Octopus Agile  →  시간 이동 행동 변화 가능 입증
미국 OhmConnect      →  가정 DR을 도매시장 자원화 가능 입증
프랑스 EDF/NILMFormer →  AI 가전 분석 상용화 가능 입증
글로벌 WattTime      →  실시간 탄소 강도 API 표준화 입증
                        ↓
한국:  모든 인프라·데이터·법제 갖춰져 있는데 통합 서비스 0
```

## 5. 한국 정책 환경 — 모든 흐름이 부합한다

### 5-1. 분산에너지 활성화 특별법 (2024.6 시행)

- 2023.6 제정·공포, 2024.6 시행
- VPP(가상발전소) 제도화 → 도매시장에서 분산자원이 대형 발전소와 동등 지위
- 분산에너지 특구 지정 → 지역 내 재생E 자가소비
- **가정용 ESS·태양광·EV가 시장 자원이 되는 시대**

### 5-2. 2026년 정부 에너지 정책 방향

> 2026년 정책 핵심: **"요금을 올릴 것인가가 아니라, 가격 신호를 어떻게 설계할 것인가"**

| 추진 정책 | 우리 서비스와의 관계 |
|---------|------------------|
| 시간대별 요금제 (TOU) 확대 | 시민용 자동 매개 도구 |
| 동적/피크 요금제 도입 | 실시간 행동 가이드 |
| 지역별 요금제 | 지역 신호를 시민에게 전달 |
| DR/VPP/V2G 시장 확대 | **가정 부문 참여 채널** ← 우리가 채울 자리 |
| 신설 아파트 AMI 한전 연계 | 데이터 활용 정당화 |

### 5-3. 기존 제도 4종을 묶기만 해도 가치

| 제도 | 현재 상태 | 통합 시 효과 |
|------|---------|-----------|
| 한전 에너지캐시백 (월 절약 3% 이상) | 166만 호 가입, 별도 신청 필요 | 자동 신청·정산 |
| 국민DR / 에너지쉼표 (시간 감축 보상) | 7,430호 (가정), 알림 받기 어려움 | 자동 알림·참여 |
| 탄소중립포인트 (5% 이상 절감 시) | 별도 가입·계산 필요 | 자동 산정·신청 |
| 에너지바우처 (취약계층 지원) | 잔액 관리 어려움 | 잔액·소진 자동 알림 |

> 신규 제도 신설 0개. 기존 제도를 한 앱으로 묶기만 해도 시민의 부담이 사라진다.

### 5-4. 한전 행보 — 시장이 움직이고 있다

- 스마트그리드 확산사업 **2026~2030 5년 연장**
- 한전 마이데이터 기반 에너지 종합 플랫폼 추진

## 6. 서비스 상세 설계

### 6-1. 사용자 여정 (5단계)

```
[Step 1] 가입 (5분)
  - 한전 마이데이터 인증 1회
  - 가정 정보 입력: 평수, 가구원 수, 보유 가전
  - 보유 유연 자원 체크: EV, ESS, 태양광, 스마트가전

[Step 2] 학습 (1주일)
  - AI가 우리 집 사용 패턴 학습
  - NILM으로 가전별 분리
  - 사용자 라이프스타일 프로파일링 (밤형/아침형, 재택/외근)

[Step 3] 청정 시간 알림 시작
  - 매일 아침: "오늘의 청정 시간 TOP 3"
  - 실시간 푸시: "지금 풍력 폭주 → 세탁기 권장"
  - 회피 알림: "오후 6시 LNG 피크 진입 → 큰 가전 자제"

[Step 4] 행동 + 자동 보상
  - 알림 따라 가전 사용 → 한전 캐시백·DR·포인트 자동 적립
  - 스마트 가전 연결 시: "허가된 가전 자동 시간 이동"
  - 매주 리포트: "이번 주 탄소 -X kg, 보상 +Y원"

[Step 5] 누적 효과 가시화
  - "올해 누적 = 소나무 OO그루 1년치"
  - "당신이 도운 재생에너지 발전량 = OO kWh"
  - "비슷한 가구 대비 환경 기여 순위 OO%"
```

### 6-2. 4대 핵심 기능

#### ① 청정 시간 알림 (Green Hour Push)

```python
# 의사 코드
for each user:
    user_profile = NILM(user.AMI_LP_data)        # 가전별 사용 시간
    flexible_load = identify_flexible(user_profile)  # 시간 이동 가능한 부하
    
    forecast = EPSIS_24h_forecast()              # 24시간 발전 믹스
    carbon_intensity = calc_carbon_per_hour(forecast)
    
    optimal_hours = top_3(carbon_intensity, low_first)  # 가장 깨끗한 3시간
    
    if user.has_planned(flexible_load):
        push("내일 오전 11시~오후 1시 태양광 폭주 - 세탁기 사용 권장")
```

알림 예시:
- "오늘 13~15시 태양광 발전량 정점. 세탁기 시간 이동 시 탄소 -68%"
- "내일 새벽 2~4시 풍력 강풍대 예상. 식기세척기/EV 충전 추천"
- "오후 7시 저녁 피크. 큰 가전 30분 미루면 8% 절약"

#### ② 자동 모드 (Auto Schedule)

스마트 가전 연결 시:
```
사용자 설정: "세탁기 자동 모드 ON, 시간 창 = 18~익일 9시"
        ↓
AI: 그 시간 창 안에서 가장 깨끗하고 저렴한 시간 자동 선택
        ↓
세탁기 API로 자동 예약 (예: 새벽 2시)
        ↓
완료 후: "세탁 완료. 평소 대비 탄소 -52%, 비용 -180원"
```

지원 가전: 세탁기·건조기·식기세척기·EV 충전·온수기·로봇청소기

#### ③ 국민DR 자동 참여 (DR Auto-Join)

전력거래소가 국민DR 발령 시:
```
DR 발령: "오후 2~5시 전력 부족, 감축 시 1,300원/kWh 보상"
        ↓
앱이 자동 푸시: "[국민DR 발령] 오후 2~5시 가전 자제 시 5,200원 적립"
        ↓
사용자: 1탭으로 참여
        ↓
실제 감축량을 AMI 데이터로 측정 → 자동 정산 → 다음달 요금 차감
```

> 현재는 사용자가 개별 신청·관리해야 해서 7,430호만 참여. 자동화하면 10만 호 단위 참여 가능.

#### ④ 가시화 + 게임화 (Impact Dashboard)

```
주간 리포트:
  "이번 주 청정 시간 참여: 12회
   탄소 감축: 14.3 kg = 소나무 2.1그루 1년치
   적립 보상: 캐시백 1,820원 + DR 5,200원 + 포인트 720점 = 7,740원
   재생에너지 활용 기여: 우리 가정 23.4 kWh가 풍력으로 충당됨"

지역 순위:
  "관악구 평균 대비 환경 기여 상위 18%"
  
챌린지:
  "월간 청정 시간 챌린지: 30회 달성 시 추가 5,000원"
```

## 7. 사용자 페르소나 시나리오 (8개)

### 페르소나 A: 30대 맞벌이 부부 (3인 가구, 30평 아파트)

- **기존**: 퇴근 후 저녁 7시에 세탁기 + 식기세척기 동시 가동 → 누진 진입
- **청정 시간 사용 후**:
  - 자동 모드 ON → 새벽 2시에 두 가전 자동 가동
  - "이번 달 청정 시간 참여 22회 → 캐시백 +12,400원 + DR +18,200원"
  - 누진 2단계 유지 → 추가 절약 28,000원
  - **연 절약 ~70만원, 탄소 -180kg**

### 페르소나 B: 1인 가구 청년 (오피스텔, 재택근무)

- **고민**: 작은 집인데 누진 3단계 진입, 환경 의식 높음
- **청정 시간 가치**:
  - 노트북·모니터 사용 패턴 분석 → "회의 없는 시간대로 빨래·청소"
  - 게임화 + 친구 챌린지 → 재미있게 참여
  - SNS 공유: "나의 환경 기여 = 친구 OO보다 32% 높음"

### 페르소나 C: 노부부 (2인, 단독주택)

- **고민**: 요금 부담, 보일러·냉장고 큼, 자녀가 멀리 살아 도움 어려움
- **청정 시간 가치**:
  - 음성 안내 + 큰 글자 모드
  - "오늘 오후 2~3시가 깨끗한 시간이에요. 세탁기 한 번 돌려보시겠어요?"
  - 자녀 알림 공유 모드: 자녀가 부모님 절감 현황 확인

### 페르소나 D: 저소득층 (에너지바우처 수급자)

- **고민**: 바우처 잔액 관리, 폭염·한파 위험
- **청정 시간 가치**:
  - 자동 옵트인 (지자체 동의 후)
  - "바우처 잔액 5.2만원, 현재 속도라면 17일 소진"
  - "청정 시간 활용 시 한전 캐시백 +5,200원 자동 적립"
  - 저효율 가전 교체 환급 알림

### 페르소나 E: 전기차 보유자 (4인 가구, 아파트)

- **고민**: EV 충전 비용, 충전 타이밍
- **청정 시간 가치**:
  - "오늘 새벽 1~5시 풍력 강풍 + 경부하 단가 → 충전 최적"
  - 자동 충전 예약 (V2G 차량은 매도까지)
  - **EV 1대 시간 이동만으로 연 18만원 절감 + CO₂ -240kg**

### 페르소나 F: 태양광 자가소비 가정

- **고민**: 발전량 변동, 잉여 전력 활용
- **청정 시간 가치**:
  - 우리 집 태양광 + 한전 계통 동시 분석
  - "오전 11시 우리 집 태양광 폭주 → 자가소비 권장"
  - 잉여 시간 자동 매도 (분산E 특별법 활용)

### 페르소나 G: 아파트 관리사무소 (B2B 확장)

- **고민**: 단지 전체 에너지 관리, 입주민 만족도
- **청정 시간 가치**:
  - 단지별 대시보드: "우리 단지 환경 기여 OO 순위"
  - 입주민 절감 챌린지 운영 도구
  - 포스코이앤씨 "에너지 쉼표 챌린지" 사례 → 우리가 인프라

### 페르소나 H: 한전·전력거래소 (공공 운영자 관점)

- **고민**: 가정 DR 참여율 0.45%, 재생E 출력제한 증가
- **청정 시간 가치**:
  - 자동화된 시민 매개 채널 → 가정 DR 참여 10배 확대 가능
  - VPP 자원으로 시민 가정 활용 → 도매시장 유연성 자원
  - 출력제한 시간에 가정 흡수 유도 → **재생E 활용률 상승**

| 항목 | My한전 | 한전 캐시백 | 국민DR | 서울 에너지 지도 | 포스코 에너지 쉼표 | **청정 시간** |
|------|-------|---------|-------|------------|--------------|------------|
| 사용자 단위 | 가구 | 가구 | 가구 | 동/구 | 단지 | **가구 + 가전 + 시간대** |
| 가전별 분석 | ❌ | ❌ | ❌ | ❌ | ❌ | **NILM AI** |
| 시간 이동 가이드 | ❌ | ❌ | 발령 시점만 | ❌ | DR 시점만 | **24시간 사전 + 자동** |
| 청정 시간 알림 | ❌ | ❌ | ❌ | ❌ | ❌ | **있음** |
| 자동 보상 | ❌ | 분기 정산 | 수동 | ❌ | ❌ | **4종 자동 통합** |
| 탄소 가시화 | ❌ | 단순 환산 | ❌ | 동 단위 | ❌ | **개인 + 시간별** |
| 게임화 | ❌ | ❌ | ❌ | ❌ | 단지 챌린지 | **개인·지역·전국** |
| 사용 동기 | 없음 | 절약 | 보상 (까다로움) | 정보 | DR 보상 | **자동 보상 + 환경 기여 + 절약** |

### 차별화 한 줄 정리

> 다른 서비스는 **"덜 써라"** 거나 **"특정 시점에 행동해라"** 를 요구한다. 청정 시간은 **"같은 가전을 다른 시간에 쓰면 자동으로 환경에 기여하고 보상받는다"** 는 새로운 행동 인식을 만든다.

## 14. 정책 제언 (사업·정책 부문 핵심)

기획서 정책 제안 5종:

### 제언 1: **국가 표준 가정 부문 청정 에너지 매개 플랫폼 지정**
- 기후에너지환경부 + 한전 + 전력거래소 공동 운영
- 한전 단독 사업이 아닌 공익 플랫폼 명시
- 분산에너지 특별법 시행령에 가정 매개 채널 조항 신설

### 제언 2: **시간대별 탄소 배출계수 공시 의무화**
- 현재는 연 평균 배출계수만 공식 발표 (0.4434 tCO₂/MWh)
- 시간대별·계절별·지역별 배출계수 공시 의무화
- 시민·기업이 행동 변화 근거 확보

### 제언 3: **국민DR 자동 참여 옵션 표준화**
- 현재 가정 7,430호에 머무는 이유 = 알림·신청·정산 모두 수동
- 마이데이터 기반 자동 참여를 한전 기본 옵션으로 제공
- 참여율 100배 확대 시 가정 부문 DR 자원 = LNG 1~2기 분량

### 제언 4: **취약계층 자동 가입 (Default Opt-In)**
- 에너지바우처 수급자 자동 가입 (동의 시)
- 자동 보상 + 잔액 알림 = 에너지 복지 자동화
- 디지털 격차 해소 효과

### 제언 5: **마이데이터 + 엣지 추론 표준 가이드**
- NILM 모델을 사용자 기기에서 실행하는 엣지 옵션 표준화
- 차등 프라이버시 + K-익명성 보장 가이드라인
- 시민 신뢰 기반 데이터 활용 모델

## 16. 기획서에 쓸 핵심 메시지 3종

### 메시지 1 (Why)

> **"AI 시대에 한국이 탄소중립을 지키려면, 발전소를 더 짓는 것이 아니라 시민의 시간을 바꿔야 합니다."**

### 메시지 2 (What)

> **"청정 시간은 가정의 AMI 데이터, 발전 믹스, 탄소 배출계수를 AI로 융합해 '지금 가전을 쓰면 가장 친환경적인 시간'을 시민에게 알리고, 참여를 자동 보상으로 정산하는 공공 플랫폼입니다."**

### 메시지 3 (Impact)

> **"가정 부문 5% 시간 이동만으로 LNG 화력 1기를 짓지 않을 수 있습니다. 우리는 시민·한전·정부가 이미 가진 데이터와 제도를 묶어 그 변화를 자동화합니다."**
